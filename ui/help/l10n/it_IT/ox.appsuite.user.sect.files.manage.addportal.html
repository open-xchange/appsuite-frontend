9.5.8. Aggiungere file al portale

È possibile aggiungere un file come riquadro del portale.

Come aggiungere un file al portale:

  • Fare clic su un file nell'area di visualizzazione. Fare clic su Altro nella finestra a comparsa. Fare clic su Aggiungi al portale nel menu.
