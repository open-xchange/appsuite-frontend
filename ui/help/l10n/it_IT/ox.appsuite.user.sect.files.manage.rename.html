9.5.4. Modificare i nomi dei file

È possibile modificare il nome di un file.

Come modificare il nome di un file:

Prerequisito: avere i permessi appropriati per modificare gli oggetti nella cartella contenente il file.

  1. Fare clic su un file nell'area di visualizzazione. Fare clic su Altro nella finestra a comparsa. Fare clic su Rinomina nel menu.

  2. Modificare il nome del file. Notare l'estensione.

  3. Fare clic su Rinomina.
