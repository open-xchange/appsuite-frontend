9.4. Creación de archivos

Un archivo puede contener:

  • una descripción

  • una o más versiones de un archivo guardado en local

  • comentarios

Cómo crear un nuevo archivo:

  1. Abra una carpeta del árbol de carpetas.

    Nota: Abra una carpeta en la que tenga los permisos adecuados para crear objetos.

  2. Presione el icono Subir nuevo archivo de la barra de herramientas. Presione en Seleccionar archivo.

    Consejo: Puede crear un nuevo archivo presionando Subir nuevo archivo en la barra de funciones de la parte superior del área de visualización.

  3. Introduzca un título en la ventana Añadir archivo nuevo.

  4. Para subir un archivo almacenado localmente, presione Seleccionar archivo. Seleccione un archivo.

  5. Puede introducir información adicional en el campo Descripción.

  6. Presione Guardar.

Consejo: También se puede crear un nuevo archivo arrastrándolo desde su escritorio hasta la ventana de la aplicación Ficheros y soltándolo allí, en la parte superior.
