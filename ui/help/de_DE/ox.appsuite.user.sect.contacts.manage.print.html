7.6.7. Kontakte drucken

Um Kontakte zu drucken, haben Sie folgende Möglichkeiten:

  • die Telefondaten eines Kontakts drucken

So drucken Sie die Telefondaten eines Kontakts:

  1. Öffnen Sie im Ordnerbaum ein Adressbuch.

  2. Wählen Sie in der Seitenleiste einen Kontakt. Klicken Sie im Anzeigebereich auf Weitere. Klicken Sie im Menü auf Drucken. Ein Fenster mit einer Druckvorschau wird geöffnet.

  3. Falls erforderlich, ändern Sie die Druckeinstellungen. Klicken Sie auf die Schaltfläche Drucken.

  4. Schließen Sie das Fenster mit der Druckvorschau.

Tipp: Sie können auch eine Telefonliste mit mehreren Kontakten drucken.
