5.2.1. Reihenfolge der Portal-Kacheln ändern

Sie können festlegen, in welcher Reihenfolge die Kacheln angezeigt werden.

So ändern Sie die Reihenfolge der Portal-Kacheln:

  1. Ziehen Sie eine Kachel an eine andere Position.

  2. Legen Sie die Kachel an der neuen Position ab.
