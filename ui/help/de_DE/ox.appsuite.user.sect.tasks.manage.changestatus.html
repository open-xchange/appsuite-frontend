11.5.5. Aufgabenbestätigung ändern

Sie können Ihre Aufgabenbestätigung nachträglich ändern.

So ändern Sie Ihre Aufgabenbestätigung:

  1. Wählen Sie in der Seitenleiste eine Aufgabe.

  2. Klicken Sie im Anzeigebereich auf Weitere. Klicken Sie im Menü auf Bestätigungsstatus ändern.

  3. Wählen Sie im Fenster einen Bestätigungsstatus. Bei Bedarf geben Sie eine Nachricht ein.

  4. Klicken Sie auf Status ändern.
