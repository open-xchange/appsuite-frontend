9.4. Dateien anlegen

Eine Datei kann die folgenden Bestandteile enthalten:

  • Eine Beschreibung

  • eine oder mehrere Versionen einer lokal gespeicherten Datei

  • Anmerkungen

So legen Sie eine neue Datei an:

  1. Öffnen Sie im Ordnerbaum einen Ordner.

    Hinweis: Öffnen Sie einen Ordner, in dem Sie die Berechtigung zum Anlegen von Objekten haben.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Hinzufügen . Klicken Sie auf Neue Datei hochladen.

  3. Geben Sie im Fenster Neue Datei hinzufügen einen Titel ein.

  4. Um eine lokal gespeicherte Datei hochzuladen, klicken Sie auf Date wählen. Wählen Sie eine Datei aus.

  5. Bei Bedarf können Sie in Beschreibung weitere Informationen eingeben.

  6. Klicken Sie auf Speichern.

Tipp: Sie können eine neue Datei auch anlegen, indem Sie eine Datei vom Desktop Ihres Betriebssystems in das Fenster der App Dateien ziehen und dort in der oberen Hälfte ablegen.
