Exporter des données

La fonctionnalité d'exportation vous permet d'exporter certaines données dans un fichier que vous pourrez exploiter dans d'autres programmes. Vous pouvez exporter les données suivantes :

  • Des contacts aux formats suivants :

    • CSV

    • vCard

    • hCard

  • Des rendez-vous aux formats suivants :

    • iCalendar

  • Des tâches aux formats suivants :

    • iCalendar

Ces formats sont standardisés et peuvent ensuite être importés dans de nombreux autres programmes. Vous pouvez :

  • exporter les données présentes dans un dossier.
