6.6.3. Inviter tous les destinataires d'un courriel à un rendez-vous

S'il y a plusieurs destinataires à un message, ils peuvent être invités en bloc à un nouveau rendez-vous.

Pour inviter tous les destinataires du message à un rendez-vous :

  1. Choisissez un courriel dans la barre latérale.

  2. Cliquez sur Tous les destinataires dans la zone d'affichage.

    Remarque : ce bouton ne s'affiche que si le message comporte plusieurs destinataires.

  3. Choisissez Inviter à un rendez-vous dans le menu.

  4. Complétez les champs pour créer un rendez-vous.
