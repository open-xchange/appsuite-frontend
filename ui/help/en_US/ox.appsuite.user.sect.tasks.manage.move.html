11.5.4. Moving tasks

You can move a task or multiple tasks at once to another folder.

How to move a task:

Prerequisite: You have the permission to create objects in the target folder.

  1. Select a task in the sidebar

  2. Click on More in the display area. Click on Move in the menu.

  3. Select a folder in the Move window. Click on OK.

Tip: In order to move tasks using drag and drop, select a task or multiple tasks in the sidebar. Drag the selected files to a folder in the folder tree. Drop them there.
