8.5.3. Changing a task's due date

You can change a task's due date and time.

How to change a task's due date:

Prerequisite: You have the permission to create objects in the folder containing the task.

  1. Select a task in the sidebar.

  2. Click on Change due date in the display area. Select an entry.
