6.6.4. Saving all recipients of an E-Mail as a distribution list

If an E-Mail contains multiple recipients, you can save all the recipients as a new distribution list.

How to save the recipients of an E-Mail as a distribution list:

  1. Select an E-Mail from the sidebar.

  2. Click on All recipients in the display area.

    Note: This button is only displayed if the E-Mail has multiple recipients.

  3. Select Save as distribution list from the menu.

  4. Complete the data for Creating a distribution list.
