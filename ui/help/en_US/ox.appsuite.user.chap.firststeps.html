What you initially should do.

  • Get familiar with the OX App Suite user interface

  • Customize the OX App Suite basic settings.

  • Check your personal contact data. Change or complete it, if required.

  • For security reasons you should change your password.

  • In order to access social network contacts e.g. Facebook contacts you need to set up accounts for social networks.
