11.5.6. Printing tasks

In order to print tasks you can do the following:

  • print a task's data

How to print a task's data:

  1. Select a task in the sidebar.

  2. Click on More in the display area. Click on Print in the menu. A window with a print preview opens.

  3. If required, change the printer settings. Click on the Print button.

  4. Close the print preview window.

Tip: You can also print the data of multiple tasks at once.
