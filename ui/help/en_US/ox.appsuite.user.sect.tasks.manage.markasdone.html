11.5.2. Marking tasks as done

You can mark a task or multiple tasks at once as done.

How to mark a task as done:

Prerequisite: You have the permission to edit objects in the folder containing the task.

  1. Select a task in the sidebar.

  2. Click on Done in the display area .
