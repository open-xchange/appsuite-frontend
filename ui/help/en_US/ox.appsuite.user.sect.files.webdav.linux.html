9.6.1. Setting up WebDAV under Linux

How to set up WebDAV access under Linux:

  1. Open the KDE Konqueror or a similar browser.

  2. Enter the following address in the address bar: webdav://<address>/servlet/webdav.infostore Replace the <address> with the Groupware Server's ip address or url.

  3. Enter your username and password for accessing the Groupware Server.

Result: The Files app files are displayed in the browser.
