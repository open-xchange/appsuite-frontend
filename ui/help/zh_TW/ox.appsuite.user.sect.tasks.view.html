11.2. 顯示任務

如何顯示任務:

  1. 在資料夾樹中 開啟 任務資料夾。

  2. 要快速尋找某個工作，使用下列一種方式:

    • 要搜尋某個主旨，使用 搜尋功能 。

    • 要排序任務清單，點選清單下方的 排序 圖示 。自選單選擇排序方式。

    • 若只要選擇到期的任務，點選清單下方的 排序 圖示 。在選單中停用 顯示已完成的任務 。

  3. 點選側邊欄中的任務。任務的資料將顯示於顯示區域。

  4. 要顯示其他任務，使用下列方法之一:

    • 點選側邊欄中的其他任務。

    • 使用方向鍵瀏覽清單。
