實現 iCal 導入

匯入 iCal 資料時注意下列項目。

  • 請確定您想要匯入的檔案包含正確的 iCal 資料。

如何導入 iCal 格式的約會或任務：

  1. 執行 行事曆 oder 任務 應用程式。

  2. 選擇 要匯入約會或任務到資料夾樹的資料夾。

  3. 單擊資料夾樹下方的 編輯 圖標 。點選 匯入。

  4. 在 匯入至 窗口中單擊 選擇檔案。選擇 iCal 格式的檔案。

  5. 單擊 匯入。

結果： 約會或任務被新增到文件夾。
