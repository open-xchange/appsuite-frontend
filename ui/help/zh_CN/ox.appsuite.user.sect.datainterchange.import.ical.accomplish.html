13.1.1.3. 实现 iCal 导入

在汇入 iCal 数据时，注意以下事项。

  • 确保您要导入的文件包含正确的 iCal 数据。

如何导入 iCal 格式的约会或任务：

  1. 执行 日历 或 任务 应用程序。

  2. 在文件夹树里选择汇入约会或任务的文件夹。

  3. 单击文件夹书下面的编辑文件夹图标。单击汇入。

  4. 在系统窗口中单击 选择文件（在汇入到窗口）。选择一个 iCal 格式的文件。

  5. 单击汇入。

结果：约会或任务被添加到文件夹。
