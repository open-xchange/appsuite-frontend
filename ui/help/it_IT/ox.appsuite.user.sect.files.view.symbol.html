Visualizzare i file nella vista Icone

Come visualizzare un file nella vista Icone:

  1. Fare clic sull'icona Vista nella barra degli strumenti. Fare clic su Icone.

  2. Selezionare una cartella dal percorso di navigazione nella parte alta dell'area di visualizzazione o aprire una cartella nell'albero delle cartelle.

  3. Per trovare rapidamente un determinato file, utilizzare la funzione di ricerca

  4. Fare clic sull'icona di un file nell'area di visualizzazione. I dati sono visualizzati nella finestra a comparsa. Per alcuni tipi di file viene mostrata un'anteprima.
