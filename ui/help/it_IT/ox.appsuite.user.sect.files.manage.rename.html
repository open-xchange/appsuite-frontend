9.5.4. Modificare i nomi dei file

È possibile modificare il nome di un file.

Come modificare il nome di un file:

Prerequisito: avere i permessi appropriati per modificare gli oggetti nella cartella contenente il file.

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic su Altro. Nel menu, fare clic su Rinomina.

    • Nella vista Elenco, fare clic su un file nella barra laterale. Fare clic su Altro nell'area di visualizzazione. Nel menu, fare clic su Rinomina.

  2. Modificare il nome del file. Notare l'estensione.

  3. Fare clic su Rinomina.
