9.5.1. Inviare file come un collegamento

È possibile inviare un collegamento a un file come allegato di posta a un utente. È possibile inviare collegamenti a più file contemporaneamente come allegati di posta. Leggere anche i suggerimenti in Sezione 9.7, «File e squadre».

Come inviare un collegamento a un file come allegato di posta:

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic su Altro. Nel menu, fare clic su Invia come collegamento.

    • Nella vista Elenco, fare clic su un file nella barra laterale. Nell'area di visualizzazione, fare clic su Altro. Nel menu, fare clic su Invia come collegamento.

  2. Nella pagina Componi un nuovo messaggio, completare i dettagli per inviare il messaggio di posta.
