9.3.3. Riprodurre i file multimediali

È possibile utilizzare le seguenti funzioni multimediali:

  • visualizzare le immagini come una presentazione

  • riprodurre i file audio

  • riprodurre i file video

Come visualizzare le immagini come una presentazione:

  1. Fare clic sull'icona Vista nella barra degli strumenti. Fare clic su Icone.

  2. Selezionare una cartella con immagini dal percorso di navigazione nella parte alta dell'area di visualizzazione. In alternativa, aprire una cartella con immagini nell'albero delle cartelle.

  3. Fare clic su Visualizza presentazione nell'area di visualizzazione. Per visualizzare la presentazione in modalità a schermo intero, fare clic su Schermo intero.

Come riprodurre i file audio:

  1. Fare clic sull'icona Vista nella barra degli strumenti. Fare clic su Icone.

  2. Selezionare una cartella con file audio dal percorso di navigazione nella parte alta dell'area di visualizzazione. In alternativa, aprire una cartella con file audio nell'albero delle cartelle.

  3. Fare clic su Riproduci file audio nell'area di visualizzazione. Si aprirà una finestra del lettore. Per riprodurre i file audio sullo sfondo, fare clic su Minimizza.

Come riprodurre file audio:

  1. Fare clic sull'icona Vista nella barra degli strumenti. Fare clic su Icone.

  2. Selezionare una cartella con file video dal percorso di navigazione nella parte alta dell'area di visualizzazione. In alternativa, aprire una cartella con file video nell'albero delle cartelle.

  3. Fare clic su Riproduci file audio nell'area di visualizzazione. Si aprirà una finestra del lettore. Per riprodurre i file audio sullo sfondo, fare clic su Minimizza.
