Inviare file come allegati di posta

È possibile inviare la versione corrente come allegato di posta. È possibile inviare le versioni correnti di più file contemporaneamente come allegati di posta.

Come inviare la versione corrente come un allegato di posta:

  1. In base alla vista selezionata, utilizzare uno dei seguenti metodi:

    • Nella vista Icone, fare clic su un file. Nella finestra a comparsa, fare clic su Invia tramite posta.

    • Nella vista Elenco, selezionare un file nella barra laterale. Fare clic su Invia tramite posta nell'area di visualizzazione.

  2. Completare tutti i dati per inviare un messaggio di posta nella pagina Componi un nuovo messaggio.
