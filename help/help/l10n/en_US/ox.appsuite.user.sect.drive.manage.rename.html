9.7.2. Editing file names

Prerequisite: You have the permission to edit objects in the folder containing the file.

How to edit the file name:

  1. Select a file in the display area.

  2. Use one of the following methods:

    • Click the More actions icon in the toolbar. Click on Rename.

    • Select Rename from the context menu.

    • Use the More actions icon in the Viewer.

    • Click on the file name on the upper left side of the Viewer.

  3. Edit the file name in the Rename window.
