5.7.9. Adding an E-Mail to the Portal

You can add an E-Mail as widget to the Portal.

How to add an E-Mail to the portal:

  1. Select an E-Mail.

  2. Click the More actions icon in the toolbar or in the detail view. Select Add to portal.

    Note: When having selected an E-Mail conversation, this function is only available in the detail view, not in the toolbar.
