Sharing

9.6.2. Accessing shared data

In the folder tree, you have access to objects shared with you by other users.

How to access objects shared by other users:

  1. Click on Shared files in the folder tree.

    If a user shared objects with you, a folder named after the user will be displayed.

  2. Open the folder to display its contents.

  3. Select one or several objects. Use the function bar entries.

    Note: Depending on whether the objects are shared with read or edit rights, different functions can be available.
