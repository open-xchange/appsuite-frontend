Managing Tasks

8.6.2. Marking tasks as done

You can mark a task or multiple tasks at once as done.

How to mark a task as done:

Prerequisite: You have the permission to create objects in the folder containing the task.

  1. Select a task from the list.

  2. Select Mark as done from the toolbar.
