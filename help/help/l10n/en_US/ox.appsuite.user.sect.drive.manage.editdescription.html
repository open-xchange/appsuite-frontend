9.7.3. Creating or editing descriptions

You can newly create or edit a file's description.

Prerequisite: You have the permission to edit objects in the folder containing the file.

How to create or edit a file's description:

  1. Select a file in the display area.

    If no details are displayed, click on View in the toolbar. Enable File details.

  2. To view the buttons, click the Panel on/off icon next to Description in the sidebar.

    In order to add a description, click on Add a description.

    In order to edit a description, click on Edit description.

You can also use the function in the sidebar of the Viewer.
