The Email Components

5.1.7. The Email pop-up

Displays the contact data of the sender or recipient that you select in the detail view.

Content

  • Buttons for frequently used functions

    • If the contact is not yet saved in an address book, Add to address book will be displayed. If clicking on the button, the window for creating a new contact opens.

    • Send email. Opens the window for sending a new email to the contact.

    • Invite. Opens the window for creating an appointment with this contact.

  • More actions icon . If clicking the icon, a menu with frequently used functions for organizing contacts is opened.

  • The contact's Halo View.
