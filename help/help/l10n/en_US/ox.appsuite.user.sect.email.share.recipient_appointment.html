E-Mails within a Team

5.7.3. Inviting all E-Mail recipients to an appointment

If there are several E-Mail recipients, all recipients can be invited to a new appointment.

How to invite all E-Mail recipients to an appointment:

  1. Select an E-Mail.

  2. Click the Actions icon in the detail view.

  3. Select Invite to appointment from the menu.

    You can also use the Actions icon in the detail view.

  4. Complete the details for creating an appointment.
