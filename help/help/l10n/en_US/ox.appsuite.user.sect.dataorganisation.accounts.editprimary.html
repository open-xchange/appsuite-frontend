11.3.1. Editing the primary email account

The following options exist:

  • Change the name of your primary email account. This name is displayed in the email settings.

  • Change the preset sender name

  • Change the folder of your primary email account

How to edit the primary email account:

  1. In the menu bar, click the Settings icon. Click on All settings.

  2. In the sidebar, click on Accounts.

  3. Click on Edit next to the primary email account in the display area. The Edit mail account window displays the current settings.

  4. To change the account name, click into the input field next to Account name.

    Note: The entries for Incoming server and Outgoing server can not be changed.

  5. To change the preset sender name, click into the input field next to Your name.

    You can overwrite this preset name when composing an email.

  6. To select other folders, browse down to Standard folders.

    Click on Select next to a folder. Select a subfolder below the INBOX folder.

    Note: Do not select the INBOX folder. Select a subfolder below INBOX.
