7.9.5. Using calendar colors

You can assign a color to a calendar. All appointments in this calendar will then be displayed in the color of the calendar. If you assign an individual color to an appointment when creating or editing it, the appointment will be displayed in the color assigned, not in the color of the calendar.

How to select a calendar color:

  1. Click the Actions icon next to the calendar name in the folder view.

  2. Select a color from the color selection drop down.
