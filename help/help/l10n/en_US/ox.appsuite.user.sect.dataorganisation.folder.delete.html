Managing Data with Folders

10.1.8. Deleting folders

You can delete subfolders in your personal folders.

Warning: If you delete a folder, all subfolders and objects in this folder will also be deleted. Deleted subfolders and objects can not be restored.

How to delete a folder:

  1. In the folder view, select the folder that you want to delete.

  2. Click the Actions icon next to the folder name. Click the Delete button.

  3. Confirm that you want to delete the folder.
