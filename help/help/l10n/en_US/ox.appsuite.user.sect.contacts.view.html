Address Book

6.2. Displaying Contacts

By default, the contacts in the Global address book are displayed. Your personal address book can be found in the folder tree below Contacts.

How to display a contact:

  1. Open an address book in the folder tree.

  2. In order to quickly find the wanted contact, use the following methods:

    • In order to only display contacts with certain properties, use the search function.

    • In order to display contacts with a certain initial letter, click a letter in the navigation bar.

  3. Click on a contact in the list. The contact's data is displayed in the detail view.

  4. In order to display another contact, do one of the following:

    • Click on another contact in the list.

    • Use the cursor keys to browse the list.
