7.1.10. The scheduling view

Shows the appointments of the participants that you add to a scheduled appointment.

In order to display the scheduling view, do one of the following:

  • Click on Scheduling in the toolbar.

  • Click on Find a free time in the appointment editing window.

Content

  • The Browse icons enable you to browse back and forth.

  • Start and end date of the selected time range and the calendar week. If clicking on it, the date picker opens that allows to set a different time range.

  • Icons for adjusting the zoom ratio for the calendar sheet view

  • Options button. If clicking on it, a menu with several functions opens:

    • Compact. Defines the line height for displaying the participants and resources

    • Show fine grid. Defines a smaller grid spacing for the time grid.

    • Free. Defines whether free appointments are displayed. Such appointments do not cause conflicts.

    • Week or Month date range. Defines the date range that is displayed in the scheduling view.

    • Hide non-working time. Defines whether times that are outside the working hours are hidden.

  • If the scheduling view was opened from the toolbar: Close icon

  • An input field for participants and resources

  • A list of participants. The following information are shown for each participant: Name, Remove icon, appointments in the calendar sheet.

  • The calendar sheet of the selected time range.

  • Scrollbar. The scrollbar allows to set the part of the time range to be displayed.

  • Save as distribution list button. Saves the added participants in a new distribution list.
