10.2.1. Chats aanmaken

U heeft de volgende mogelijkheden:

  • maak een privé chat met één enkel contactpersoon

  • maak een groepschat met specifieke deelnemers

  • maak een openbaar kanaal waar andere contacten kunnen deelnemen

Een nieuwe chat maakt u als volgt:

  1. Klik op het Nieuw pictogram boven de chatlijst of op Nieuwe chat.

    Een menu opent.

  2. Gebruik één van de volgende methoden:

    • Om een chat met één contact te starten kiest u Privé chat. Het Start een nieuwe conversatie scherm opent.

      Selecteer een contact.

      Klik op Start conversatie.

    • Om een chat met meerdere deelnemers te starten kiest u Groeps-chat. Het Groeps-chat aanmaken scherm opent.

      Geef een groepsnaam. U kan een afbeelding toevoegen door op het pictogram te klikken.

      Voeg contacten toe.

      Klik op Chat aanmaken

    • Om een openbaar kanaal te maken waar andere contacten aan kunnen deelnemen, selecteert u Kanaal. Het Maak een nieuw kanaal scherm opent.window opens.

      Geef een naam van het kanaal. U kan een afbeelding toevoegen door op het pictogram te klikken.

      Klik op Kanaal aanmaken

De nieuwe chat wordt getoond in de lijst.
