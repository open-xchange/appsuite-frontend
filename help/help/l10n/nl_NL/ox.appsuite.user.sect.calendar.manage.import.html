7.8.7. Afspraken importeren

U kan afspraken importeren vanuit bestanden in het iCal formaat.

Importeren van afspraken vanuit een iCal bestand gaat als volgt:

  1. Selecteer de agenda voor het importeren van de afspraken uit het mappenoverzicht.

  2. Klik op het Acties pictogramnaast de agenda. Klik op Importeren.

  3. In het Importeer vanuit bestand scherm klik op Upload bestand. Selecteer een bestand in het iCal formaat.

    Om ook afspraken te importeren die de zelfde id hebben als reeds bestaande afspraken schakelt u Negeer bestaande afspraken in.

  4. Klik op Importeren.

Resultaat: De contacten worden toegevoegd aan de agenda.
