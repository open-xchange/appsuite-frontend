9.5.3. Mostrare il collegamento a un file

È possibile mostrare il collegamento a un file che viene salvato nell'applicazione File.

Come mostrare il collegamento a un file:

  • Fare clic su un file nella vista dettagliata. Nella finestra a comparsa, fare clic sull'icona . Fare clic su Mostra collegamento interno nel menu.

    È possibile selezionare anche uno o più file. Fare clic sull'icona Altro nella barra degli strumenti. Fare clic su Mostra collegamento interno nel menu.
