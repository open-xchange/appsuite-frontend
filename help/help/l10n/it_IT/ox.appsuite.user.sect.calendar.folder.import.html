7.7.3. Importare calendari da file

È possibile importate i calendari che sono stati creati all'interno di altre applicazioni. Il calendario che si vuole importare deve essere disponibile come file iCal.

Come importare calendari da un file:

  1. Nella vista delle cartelle, fare clic su Aggiungi nuovo calendario. Si aprirà un menu.

    Fare clic su Carica file. Si aprirà la finestra Importa da file.

  2. Fare clic suCarica file. Selezionare un file.

  3. Digitare un nome per il nuovo calendario. Fare clic su Importa.
