9.8. Utilizzare le archiviazioni sul cloud

Se si utilizzano archiviazioni sul cloud come Google Drive, Dropbox, Box o OneDrive, è possibile accedere alle proprie archiviazioni dall'interno del groupware. Sono disponibili le seguenti opzioni.

  • Prima è necessario configurare e modificare l'archiviazione sul cloud nel groupware.

  • È quindi possibile utilizzare la propria archiviazione sul cloud con l'aiuto dell'albero delle cartelle.
