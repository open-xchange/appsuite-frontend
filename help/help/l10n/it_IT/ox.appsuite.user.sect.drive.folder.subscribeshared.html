9.4.2. Sottoscrivere cartelle pubbliche e condivise di File

È possibile definire quali cartelle pubbliche e condivise di File sono visualizzate nella vista delle cartelle sottoscrivendo tali cartelle.

Come definire le cartelle di File che si desidera sottoscrivere:

  1. Nella vista delle cartelle, fare clic su Gestisci condivisioni.

    Si aprirà una finestra. La finestra mostra le cartelle pubbliche e condivise di File .

  2. Se si vuole che una cartella sia visualizzata, fare clic sul pulsante Sottoscrivi cartella .

    Se non si desidera più che una cartella sia visualizzata e sincronizzata, disabilitare il pulsante Sottoscrivi cartella.
