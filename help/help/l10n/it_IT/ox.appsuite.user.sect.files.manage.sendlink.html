Gestire i file

9.5.1. Inviare file come un collegamento

È possibile inviare un collegamento a un file come allegato di posta a un utente. È possibile inviare collegamenti a più file contemporaneamente come allegati di posta.

Come inviare un collegamento a un file:

  1. Fare clic su un file nella vista dettagliata. Nella finestra a comparsa, fare clic sull'icona Altro. Fare clic su Invia come collegamento interno nel menu.

    È possibile selezionare anche uno o più file. Fare clic sull'icona Altro nella barra degli strumenti. Fare clic su Invia come collegamento interno nel menu.

  2. Completare tutti i dati per inviare un messaggio di posta nella pagina Componi un nuovo messaggio.
