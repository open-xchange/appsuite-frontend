10.2. Condividere dati

È possibile collaborare con utenti interni o collaboratori esterni condividendo i propri dati del groupware con permessi di lettura o modifica:

Sono disponibili le seguenti opzioni:

  • Condividere i dati

  • Modificare le condivisioni

  • Inviare nuovamente un invito

  • Rimuovere le condivisioni

  • Visualizzare o modificare le condivisioni nell'applicazione File

  • Sottoscrivere le cartelle di posta elettronica condivise o personali

  • Accedere alle condivisioni degli altri utenti
