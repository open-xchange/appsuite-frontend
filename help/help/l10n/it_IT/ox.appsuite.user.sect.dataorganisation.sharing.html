Organizzazione dei dati e condivisioni

11.2. Condividere dati

Puoi collaborare con utenti interni o collaboratori esterni condividendo i seguenti dati con autorizzazioni di lettura o modifica:

  • Cartelle di posta elettronica, rubriche, calendari, cartelle delle attività

    La rubrica globale non può essere condivisa.

  • Cartelle di File, file

Sono disponibili le seguenti opzioni:

  • Condividere i dati

  • Modificare le condivisioni

  • Inviare nuovamente un invito

  • Rimuovere le condivisioni

  • Visualizzare o modificare le condivisioni nell'applicazione File

  • Sottoscrivere le cartelle di posta elettronica condivise o personali

  • Accedere alle condivisioni degli altri utenti

Informazioni sui permessi che possono essere utilizzati quando si condivide sono disponibili qui:
