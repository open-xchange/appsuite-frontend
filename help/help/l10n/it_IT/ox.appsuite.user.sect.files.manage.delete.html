9.5.9. Eliminare i file

È possibile eliminare un file o più file contemporaneamente. Sono disponibili le seguenti opzioni:

  • Eliminare singoli file. I file sono spostati nella cartella File eliminati.

  • Ripristinare i file eliminati dalla cartella File eliminati.

  • Eliminare definitivamente i file nella cartella File eliminati.

    Avviso: un file eliminato nella cartella File eliminati non può essere più recuperato.

Come eliminare un file:

  1. Fare clic su un file nella vista dettagliata. Fare clic su Elimina nella finestra a comparsa.

    È possibile selezionare anche uno o più file. Fare clic sull'icona Elimina nella barra degli strumenti.

  2. Confermare che si desidera eliminare il file.

Risultato: il file sarà eliminato.

Come ripristinare i file eliminati:

  1. Aprire la cartella File eliminati nell'albero delle cartelle.

  2. Selezionare uno o più file.

  3. Fare clic sull'icona Altro nella barra degli strumenti. Selezionare Sposta dal menu.

  4. Selezionare una cartella nella finestra Sposta. Fare clic sul pulsante Sposta.

Risultato: i file saranno spostati nella cartella selezionata.

Come eliminare un file definitivamente:

Avviso: i file eliminati definitivamente non possono essere recuperati. Prima di eliminare un file in modo definitivo, assicurarsi che il file non serva più.

  1. Aprire la cartella File eliminati nell'albero delle cartelle.

  2. Selezionare uno o più file.

  3. Fare clic sull'icona Elimina

  4. Confermare che si desidera eliminare i file.

Risultato: i file saranno eliminati definitivamente.
