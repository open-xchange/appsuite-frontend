Attività

8.6. Gestire le attività

Alcune delle tecniche per organizzare le attività richiedono la configurazione delle proprie cartelle di attività. Informazioni sulla creazione delle cartelle sono disponibili in Cartelle.

Esistono le seguenti opzioni:

  • Modificare le attività

  • Marcare le attività come completate

  • Cambiare la data di scadenza di un'attività

  • Spostare le attività

  • Cambiare le conferme delle attività

  • Stampare le attività

  • Eliminare le attività

  • Modificare più attività contemporaneamente
