9.4. Gestire i file

Alcune delle tecniche per gestire i file richiedono la preventiva configurazione delle proprie cartelle. Informazioni sulla creazione delle cartelle sono disponibili in Creare file o cartelle.

Esistono le seguenti opzioni:

  • Inviare file come un collegamento

  • Inviare file come allegati di posta

  • Mostrare il collegamento a un file

  • Modificare i nomi dei file

  • Creare o modificare descrizioni

  • Spostare file o cartelle

  • Copiare i file

  • Aggiungere file al portale

  • Eliminare i file

  • Bloccare o sbloccare i file

  • Lavorare con le versioni
