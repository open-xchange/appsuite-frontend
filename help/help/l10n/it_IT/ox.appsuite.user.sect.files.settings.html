9.8. Impostazioni dei file

Come utilizzare le impostazioni dei file:

  1. Fare clic sull'icona Menu di sistema sul lato destro della barra dei menu. Fare clic su Impostazioni nel menu.

  2. Fare clic su File nella barra laterale.

  3. Cambiare le impostazioni.

Sono disponibili le seguenti impostazioni.

  • Vista predefinita

Vista predefinita Specifica la vista predefinita per i file.
