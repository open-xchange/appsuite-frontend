9.9. File e squadre

È possibile condividere i propri file con gli utenti interni. In base alle necessità, possono essere applicati diversi metodi.

  • In File pubblici e in File condivisi, è possibile trovare cartelle di file condivise da altri utenti.

  • Per rendere disponibili file aggiuntivi agli utenti interni o ai collaboratori esterni, procedere come segue:

    • Creare una nuova cartella personale o pubblica e copiare o spostare i file richiesti in questa cartella.

    • >Condividere questa cartella.

    È inoltre possibile condividere una cartella esistente.
