9.2. Visualizzare i file

In modo predefinito, viene visualizzato il contenuto della cartella personale dei file. Il nome della cartella è I miei file.

Come visualizzare un file:

  1. Fare clic su Vista nella barra degli strumenti. Selezionare una delle seguenti voci: Elenco, Icone, Riquadri.

  2. Se l'albero delle cartelle è chiuso, fare clic su Vista nella barra degli strumenti. Abilitare Vista della cartella.

    Aprire una cartella nell'albero delle cartelle.

  3. Facendo clic su un file nell'area di visualizzazione. I dati sono visualizzati nella finestra a comparsa.

Suggerimento: per trovare rapidamente un determinato file, utilizzare la funzione di ricerca.
