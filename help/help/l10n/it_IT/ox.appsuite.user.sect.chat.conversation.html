10.2. Lavorare con le chat

Scoprire come utilizzare le chat per scambiare messaggi con singole persone o gruppi.

Prima di poter inviare messaggi, è necessario determinare con chi si desidera scambiare messaggi. Per farlo, creare nuove chat o partecipare a quelle esistenti. Sono disponibili le seguenti opzioni:

  • Creare chat

  • Modificare le chat

  • Partecipare a una chat

  • Aggiungere chat ai preferiti

  • Chiudere una chat

  • Abbandonare una chat
