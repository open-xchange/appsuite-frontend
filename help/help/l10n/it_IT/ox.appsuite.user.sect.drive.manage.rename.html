9.7.2. Modificare i nomi dei file

Prerequisito: avere i permessi appropriati per modificare gli oggetti nella cartella che contiene il file.

Come modificare il nome di un file:

  1. Selezionare il file nell'area di visualizzazione.

  2. Utilizzare uno dei seguenti metodi:

    • Fare clic sull'icona Altre azioni nella barra degli strumenti. Fare clic su Rinomina.

    • Selezionare Rinomina dal menu contestuale.

    • Utilizzare l'icona Altre azioni nel visualizzatore.

    • Fare clic sul nome del file in alto a sinistra del visualizzatore.

  3. Modificare il nome di un file nella finestra Rinomina.
