12.7. Guard 設定

如何使用 Guard 設定：

  1. 點擊選單列右邊的設定圖示 。

  2. 點擊側邊欄的安全性。點擊 Guard。

將顯示 Guard 安全性設定 頁面。

有以下選項：

設定選項

  • 撰寫郵件時預設在寄出郵件時加密. 設定是否預設寄出加密郵件。

  • 預設在外寄郵件中加入簽章. 設定是否預設寄出加密郵件。

  • 為新郵件預設使用行內 PGP. 設定是否直接在行內使用 PGP 加密。僅只在收件者的郵件軟體不支援 PGP 時使用該設定。若使用行內 PGP，收件者還是能解密訊息。若您使用此選項，將無法以 HTML 格式寄出郵件。

  • 記住密碼預設值. 設定 Guard 要記住密碼的時間範圍預設值。當 Guard 向您詢問密碼時，可以變更此預設設定。

密碼管理

  • 包含變更或重設密碼的功能

  • 參見密碼管理

進階

  • 進階設定. 若勾選了顯示進階設定，將可以調整下列設定與功能。

    • 刪除密碼恢復資訊按鈕。

      若要在必需時重設密碼，需在伺服器上儲存您加密過的密碼。若您不想在伺服器上儲存密碼，可以完全移除重設密碼的功能。

      參見密碼管理

    • 電子郵件快取

      設定解密的郵件是否要在瀏覽器開啟時用加密格式顯示。

    • 金鑰段落

      管理自己的金鑰、公鑰或 Autocrypt 金鑰

      參見金鑰

    • Autocrypt 段落

      設定對來信使用 Autocrypt。讓您可以轉移 Autocrypt 金鑰到其他客戶端。

      參見使用 Autocrypt 加密郵件通訊

    • 傳送加密郵件時，有更多選項。

      參見傳送加密郵件

金鑰

通常在收發加密郵件時，不需要使用金鑰管理功能。這些功能在下列情況下非常有用：

  • 您想要在其他郵件軟體，例如本機郵件客戶端使用您的 Guard PGP 金鑰時。

  • 您有從其他 PGP 應用程式打造的 PGP 金鑰時。您想要在 Guard 使用那些金鑰時。

  • 您有外部夥伴的公鑰時。要傳送加密訊息給此位外部夥伴，或在無法存取金鑰伺服器時驗證收到的訊息，想要直接匯入該夥伴的公鑰到 Guard 時。

  • 您想要提供公鑰給某位外部收件者，讓他們可以傳送加密訊息給您，而不需要從金鑰伺服器下載時。

有下列選項：

  • 下載我的公鑰 按紐. 將公鑰儲存至您的電腦。

  • 您的金鑰 按鈕。. 開啟您的金鑰視窗，管理您 擁有的金鑰。

  • 收件者的公鑰 按鈕. 開啟您的金鑰視窗，管理您 擁有的公鑰。
