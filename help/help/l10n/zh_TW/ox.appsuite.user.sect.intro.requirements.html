Groupware 使用入門

2.2. 系統需求

要能使用 Groupware，您的電腦必須滿足以下系統需求。

解析度/螢幕大小

顯示器最小解析度為 1024 x 768。

瀏覽器

  • Microsoft Internet Explorer 10、11、Edge

  • Mozilla Firefox，最新版本

  • Mac OS X 上的 Apple Safari，目前版本

  • Google Chrome，最新版本

瀏覽器設定

  • 必須啟用 Cookies

  • 必須啟用 JavaScript

  • 必須允許彈出視窗
