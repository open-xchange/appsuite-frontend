11.7.3. 使用安全金鑰驗證登入

驗證流程如下：

  • 以 USB 或無線方式將安全金鑰連結至您的電腦。

  • 登入時，將提示您開啟安全金鑰，例如按下金鑰上的按鈕。

註：視瀏覽器而定，可能不支援此驗證功能。

如何使用安全金鑰確認身分：

  1. 將安全金鑰連結到您的電腦。

  2. 使用目前的帳號密碼登入 Groupware。

  3. 若設定過多種驗證方法，將開啟選擇兩階段驗證方式視窗。選擇 Yubikey、Google Security Key 或 FIDO 相容裝置。

    將開啟確認代碼視窗。

    開啟安全金鑰。
