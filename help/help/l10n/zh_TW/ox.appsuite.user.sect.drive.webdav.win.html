雲端空間

使用 WebDAV 存取檔案

9.8.2. 在 Windows 設定 WebDAV

如何在 Windows 下設定 WebDAV 存取：

  1. 在 Windows 檔案總管開啟 本機。

    點選側邊欄頂端的 電腦 分頁。點選新增網路地址。將開啟新增網路地址對話窗。單擊 下一步。

  2. 選擇 選擇自訂網路地址。單擊 下一步 。

  3. 在網際網路或區域網路地址欄位輸入下列地址。

    webdav://<伺服器網址>/servlet/webdav.infostore

    將 <伺服器網址> 取代為 Groupware 伺服器的 IP 或網址。單擊 下一步。

  4. 輸入用於登入 Groupware 伺服器的用戶名和密碼。點擊確定。

  5. 在下一頁中您可以向網絡資源指定一個名稱。單擊 下一步。

  6. 單擊 完成。

結果：您可以在這台電腦下方存取 雲端空間 應用程式檔案。
