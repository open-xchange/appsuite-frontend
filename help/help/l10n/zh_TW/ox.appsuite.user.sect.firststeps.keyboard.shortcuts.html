鍵盤輸入

2.5.1. 按鍵與按鍵組合

父主題：使用者介面的一般描述

預設有下列按鍵組合。視系統而定，可能有更多非官方支援的預設按鍵組合可用。

[a]

將您在清單中選擇的郵件移動到封存信件匣。

[Ctrl]+[a]

選擇清單中的所有物件。

[Del]、[Backspace]

刪除選擇的物件。

[End]

將游標移動到輸入欄位結尾。

[Enter]

執行選擇的功能。

[Esc]

關閉彈出視窗。取消視窗。

[Home]

將游標移動到輸入欄位開頭。

[Page up]、[page down]

瀏覽資料夾畫面或側邊欄。

瀏覽行事曆畫面或日曆，視顯示的時間範圍而定。

[Tab]、[Shift]+[Tab]

選擇功能、輸入欄位或選取盒。

[空白鍵]

勾選或取消勾選選擇的選取盒。

上下方向鍵

於應用程式啟動器選擇程式。

在資料夾畫面、側邊欄或顯示區域中選擇元素。

在行事曆畫面或日曆中移動游標。

開啟或關閉資料夾畫面中的資料夾。

選擇選單項目。

下方向鍵

開啟選單。

在 Windows 與 Linux 系統使用 [Ctrl]+[F6]，在 MacOS 系統使用 [F6]

切換選單列、資料夾樹、側邊欄及顯示區域。

左右方向鍵

在輸入欄位中左右移動游標。

在行事曆畫面或日曆中移動游標。

開啟或關閉資料夾畫面中的資料夾。

選擇工具列中的功能。
