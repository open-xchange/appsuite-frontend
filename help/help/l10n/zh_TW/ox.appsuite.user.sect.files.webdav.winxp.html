雲端硬碟

使用 WebDAV 存取檔案

9.7.2. 在 Windows XP 下存取 WebDAV

如何在 Windows XP 下用 WebDAV 訪問：

  1. 點選 網路上的芳鄰。在 我的電腦 視窗單擊 網路上的芳鄰 或雙擊桌面上的 網路上的芳鄰 圖示。

  2. 前往網絡任務，打開網路上的芳鄰對話窗口，然後單擊新增網路位置。一個用於添加網絡資源的嚮導啟動。單擊 下一步。

  3. 在下一個對話窗口中選擇 選擇另一個網絡位置。

  4. 輸入以下地址：https://<address>/servlet/webdav.infostore 其中 <address> 需要用組件的 IP 或網址代替。

  5. 輸入用於登錄組件伺服器的用戶名和密碼。

  6. 在下一個對話窗口中，您可以向網絡資源指定一個名稱。

    註： 如果您已經將您的編碼設置為 ISO，請不要在文件名或資料夾名中使用任何特殊字符。組件伺服器使用 UTF-8 編碼。

  7. 在下一個對話窗口中單擊 完成 。

結果： 您可以使用網路上的芳鄰存取您的雲端硬碟 應用程式檔案。
