使用 WebDAV 存取檔案

9.7.3. 在 Windows Vista 下存取 WebDAV

透過下列方式在 Windows Vista 設定 WebDAV 存取：

  • 使用 OX Updater。可在 OX 更新器的文件中找到相關信息。

  • 首先安裝微軟的網頁資料夾 軟體更新。然後設定 WebDav 存取。

如何為 WebDAV 安裝 Microsoft 軟體更新：

  1. 打開 Web 瀏覽器。

  2. 輸入以下地址：http://support.microsoft.com/kb/907306

  3. 瀏覽到 更多信息，然後單擊 Download the Software Update for Web Folders package now。

  4. 在下載頁面中單擊 Download。將下載文件 Webfldrs-KB907306-ENU.exe。

  5. 雙擊該文件。安裝將啟動。

如何在 Windows Vista 下設定 WebDAV 存取：

  1. 在 Windows 資源管理器中打開 計算機 。

  2. 在資源管理器的右窗格中右擊一個空白區域。從下拉菜單中選擇 添加一個網絡位置 。添加網絡位置 對話窗口打開。單擊 下一步。

  3. 選擇 選擇自定義網絡位置。單擊 下一步 。

  4. 在 新增網絡位置 對話窗口中輸入以下地址： https://<address>/servlet/webdav.infostore 其中 <address> 必須要用組件伺服器的 IP 或網址代替。

  5. 輸入用於登錄組件伺服器的用戶名和密碼。

  6. 在下一個對話窗口中，您可以向網絡資源指定一個名稱。

  7. 在下一個對話窗口中單擊 完成 。

結果： 您可以在 電腦 下存取您的Drive 應用程式檔案。
