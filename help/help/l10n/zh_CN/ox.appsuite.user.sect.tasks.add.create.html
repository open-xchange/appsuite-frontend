8.4.1. 创建新任务

如何创建新任务：

  1. 在文件夹视图里打开一个任务文件夹。

    备注：打开您有权限创建任务的文件夹。

  2. 在工具条里单击新建任务。

  3. 输入主题。如有需要，输入一个描述。

  4. 单击创建。

选项：

  • 要显示完整表格，单击扩张表格。以下可选功能可用：

    • 设置任务的开始和截止日期。

    • 要获得一个约会提醒，选择提醒里一个设定。对应的日期和时间会输入到通知日期里。

      要使用自定值，在通知日期里设置它们。如是者，手动输入将会在提醒字段里预选。

    • 如果不要显示主题给其他用户，打开私人。私人任务会标记私人图标。

    • 为了添加像发单信息的详细信息，单击显示详情。输入所需数据。
