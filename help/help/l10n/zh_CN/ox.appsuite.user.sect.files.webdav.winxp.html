用 WebDAV 访问文件

9.7.2. 在 视窗 XP 下设置 WebDAV

如何在 Windows XP 下设置 WebDAV 访问：

  1. 单击网上邻居。在我的计算机窗口里单击我的网路邻居或双击在视窗桌面我的网路邻居图标。

  2. 前往网络任务，打开网上邻居对话窗口，然后单击添加一个网上邻居 。一个用于添加网络资源的向导启动。单击下一步。

  3. 在下一个对话窗口中选择选择另一个网络位置 。

  4. 输入以下地址： https://<地址>/servlet/webdav.infostore其中<地址>需要用组建服务器的 IP 地址或 URL 代替。

  5. 输入组件服务器的用户名和密码。

  6. 在下一个对话窗口中，您可以向网络资源指定一个名称。

    备注：如果您已经将您的编码设置为 ISO，请不要在文件夹或文档名称中使用任何特殊字符。组建服务器使用 UTF-8 编码。

  7. 在下一个对话窗口中单击 完成。

结果：您可以使用网上邻居访问您的 磁盘 应用程序文件。
