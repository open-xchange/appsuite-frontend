9.6.1. 分享文件和文件夹

有以下选项：

  • 您能创建一个公共链接，以分享一个有读取权的文件或文件夹。

    您能将此链接给予内部用户或外部伙伴。每一位获得此链接的人员，能够检视文件或文件夹内容。

  • 以电子邮件邀请内部用户或外部伙伴到一个分享。您定义此分享的哪些权限。

    邀请电子邮件包含用于访问分享的笔记。如果一个外部伙伴访问分享，他将会自动登录成访客用户。

    备注：邀请电子邮件只会以 html 格式发送。

备注：为了分享对象，您需要有管理员权限。

如何以一个公共链接分享带有读取访问的对象：

  1. 在文件夹树里选择一个文件夹。在显示区域里选择一个对象。

    备注：选择单一对象。多个对象不能一起在一个公共链接里分享。

  2. 在工具条里单击分享图标。单击获取一个链接。

    将会打开分享对象的窗口。包括一个读取权分享的公共链接。

  3. 您可以进行以下几种选项：

    • 为了直接以电子邮件发送链接，输入相应的电子邮件地址。您能为收件人输入一个消息。

    • 默认时，数据会以无限期的读取访问分享。要为分享的数据设置一个时限，启用过期于。选择一个时限。

    • 要以一个密码保护访问，启用必须密码。输入一个密码。如果您以电子邮件发送公共链接，邮件会包括密码。

  4. 单击完成。

您也能在检视器里分享单一个文件。

您也能单击文件夹树里文件夹名称旁边的文件夹指定动作图标，以分享一个文件夹。单击获取一个链接。

为了检视一个已分享对象的权限，在显示区域里选择该对象。在详情下面，单击分享旁边的按钮。

如何邀请内部用户或外部伙伴到一个分享：

  1. 在文件夹树里选择一个文件夹。在显示区域里选择一个或多个对象。

  2. 在工具条里单击分享图标。单击邀请人员。

    会打开一个窗口。当前权限将会以列表显示。

  3. 在添加人员里输入一个电子邮件地址。电子邮件地址将会添加到列表成内部用户、群组、或访客。已批准预设权限。

    如有需要，输入讯息 。

  4. 要编辑权限，您有以下选项。

    • 要调整一个用户角色，在当前角色里单击项目。从菜单选择一个项目。

    • 要调整权限详情，在详细访问权里单击项目。从菜单里选择一个权限。

  5. 如果您分享一个文件夹，您能传送文件夹的权限到所有存在或新创建的子文件夹。为此，启用套用到所有子文件夹。

  6. 单击 保存。

您也能在检视器里分享单一个文件。

您也能单击文件夹树里文件夹名称旁边的文件夹指定动作图标，以分享一个文件夹。单击邀请人员。

为了检视一个已分享对象的权限，在显示区域里选择该对象。在详情下面，单击分享旁边的按钮。
