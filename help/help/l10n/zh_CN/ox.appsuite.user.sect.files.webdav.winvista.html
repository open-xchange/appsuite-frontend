用 WebDAV 访问文件

9.7.3. 在 视窗 Vista 下设置 WebDAV

为了设置在 Windows Vista 下设置 WebDAV 访问，请使用以下选项之一：

  • 使用 OX 更新器。可在 OX 更新器用户文件里找到说明。

  • 首先，为网页文件夹安装微软 软件更新。然后，设置WebDav 访问。

如何为 WebDAV 安装 Microsoft 软件更新：

  1. 打开 Web 浏览器。

  2. 输入以下地址：http://support.microsoft.com/kb/907306

  3. 浏览到更多信息后单击现在为网页文件夹下载软件更新。

  4. 在下载页面里单击下载。将下载文件 Webfldrs-KB907306-ENU.exe。

  5. 双击该文件。安装将启动。

如何在 Windows Vista 下设置 WebDAV 访问：

  1. 在 Windows 资源管理器中打开计算机。

  2. 在资源管理器的右窗格中右击一个空白区域。从下拉菜单中选择添加一个网络位置。添加网路位置对话窗口打开。单击下一步。

  3. 选择选择自定义网络位置 。单击继续。

  4. 在添加网络位置对话窗口中输入以下地址： https://<地址>/servlet/webdav.infostore 其中<地址>必须要用组建服务器的 IP 地址或 URL 代替。

  5. 输入用于访问组件服务器的用户名和密码。

  6. 在下一个对话窗口中，您可以向网络资源指定一个名称。

  7. 在下一个对话窗口中单击 完成。

结果：在计算机下面，您可以访问您的 磁盘 应用程序文件。
