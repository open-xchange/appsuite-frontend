12.1.2. 导入 vCard 格式的联系人

在汇入 vCard 数据时，注意以下事项。

  • 确保要导入的档案包含正确的 vCard 数据。

  • 将只会汇入每一个联系人的一个私人地址和一个业务地址。如果汇入的 vCard 档案包含额外私人和业务地址，哪些地址会在汇入时忽视。

如何导入 vCard 格式的联系人：

  1. 执行 地址簿 应用程序。

  2. 在文件夹树里选择导入联系人的文件夹。

  3. 单击文件夹名称旁边的文件夹指定动作图标。单击导入。

  4. 选择vCard格式（在汇入到窗口）。单击选择档案。选择一个 vCard 格式的文件。

  5. 单击导入。

结果： 联系人被添加到文件夹。
