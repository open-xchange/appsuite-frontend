11.1.5. Tworzenie folderów

W folderze osobistym można utworzyć dowolną liczbę dalszych podfolderów. Aby utworzyć podfoldery w folderze udostępnionym lub publicznym, musisz mieć odpowiednie uprawnienia. To, czy można tworzyć foldery, czy podfoldery, zależy od aplikacji.

Jak utworzyć nowy podfolder w aplikacji E-mail lub Pliki:

  1. W drzewie folderów wybierz folder, w którym chcesz utworzyć nowy podfolder.

    Uwaga: wybierz folder, w którym masz uprawnienia do tworzenia zadań.

  2. Kliknij ikonę Działania specyficzne dla folderu obok nazwy folderu. Kliknij przycisk Nowy podfolder.

  3. W oknie Nowy podfolder wpisz nazwę i kliknij przycisk Dodaj.

Jak utworzyć nowy folder w aplikacjach Książka adresowa, Kalendarz, Zadania:

  1. W drzewie folderów kliknij pozycję Nowy folder prywatny lub Nowy folder publiczny.

  2. W oknie Nowy folder wpisz nazwę i kliknij przycisk Dodaj folder.
