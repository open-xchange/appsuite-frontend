7.8.3. Importing calendars from files

You can import calendars that have been created from within other applications. The calendar you want to import needs to be available as an iCal file.

How to import calendars from a file:

  1. Click on Add new calendar in the folder view. A menu opens.

    Click on Upload file. The Import from file window opens.

  2. Click on Upload file. Select a file.

  3. Enter a name for the new calendar. Click on Import.

Result: A new calendar will be added.

Note: All participants of the imported appointments will be removed. Instead, you will be added as participant.
