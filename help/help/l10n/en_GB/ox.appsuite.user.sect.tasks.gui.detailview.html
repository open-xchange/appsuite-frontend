8.1.5. The Tasks detail view

Shows the data of the task that you selected in the list. If double-clicking on the task in the list, the data are displayed in a window.

Content

  • Icon for the task's priority, if set.

    Subject

    Private tasks are marked with the Private icon .

  • If set: due date, reminder date

  • status and progress

  • further task data, if set:

    • attachments

    • definition

    • For recurring tasks: repetition parameters

    • the task’s start date

    • billing information

    • The names of the participants and the external participants

      If clicking on a name, a pop-up opens. It displays the participant's Halo View.
