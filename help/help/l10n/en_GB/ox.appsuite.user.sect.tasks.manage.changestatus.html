8.9.4. Changing task confirmations

You can change your task confirmation at a later point in time.

How to change your task confirmation:

  1. Select a task from the list.

  2. Click the More actions icon in the tool bar. Click on Change confirmation status.

  3. A window opens. Select a confirmation status. If required, enter a message.

  4. you can enter a comment in the Change confirmation status window. Click on one of the buttons Decline, Tentative or Accept.
