8.8.4. Changing tasks confirmations

You can change your task confirmation at a later point in time.

How to change your task confirmation:

  1. Select a task from the list.

  2. Click the More actions iconin the tool bar. Click on Change confirmation status.

  3. A window opens. Select a confirmation status. If required, enter a message.

  4. Click on Change status.
