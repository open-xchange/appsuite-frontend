5.7.3. Marking email messages as read or unread

Unread email messages are marked with the Unread icon . The following options are available:

  • mark single E-Mail messages as read or unread

  • mark all E-Mail messages in an E-Mail folder as read

How to mark an E-Mail as unread or read:

  1. Select one or several E-Mail messages.

  2. Click the More actions icon in the tool bar or in the detail view. Select Mark as unread or Mark as read.

    Or:

    • For individual email messages, click the Mark as unread icon or the Mark as unread icon .

    • Use the context menu in the E-Mail list.

Tip: To mark all E-Mail messages in a folder as read, click on All above the list. Select Mark all E-Mail messages as read.
