9.7.10. Locking or unlocking files

When editing a file, you can lock the file. The lock has the following purposes:

  • The file lock informs other users that the file is currently being edited and might no longer be up to date.

  • If the file is located in a folder that you share with other users whom you granted edit permissions, those users can not edit the locked file.

  • If you are the owner of a shared file and another user locks the file, you can always unlock it.

How to lock files:

  1. Select one or several files in the display area.

  2. Click the More actions icon in the tool bar. Click on Lock.

How to unlock files:

  1. Select one or several locked files in the display area.

  2. Click the More actions icon in the tool bar. Click on Unlock.
