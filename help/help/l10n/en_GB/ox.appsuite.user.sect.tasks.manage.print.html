8.9.7. Printing tasks

You can print the data of a single task or of multiple tasks.

How to print a task's data:

  1. Select a task or multiple tasks from the list.

  2. Click the More actions icon in the tool bar. Click on Print. A window with a print preview opens.

  3. If required, change the printer settings. Click on the Print button.

  4. Close the print preview window.
