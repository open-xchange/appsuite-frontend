9.1.3. The Drive navigation bar

The navigation path is located below the tool bar. The following functions are available:

  • navigate within the folder structure

  • sort and select objects

Functions

  • Navigation path. It shows the path to the folder opened. To open a parent folder, click on a path entry.

  • Sort by button. Sorts the files in the display area by various criteria.

  • Select button. Contains functions for selecting or filtering objects in the display area:

    • select all files and folders, select all files, clear the selection

    • display certain file types only, display all file types
