9.7.6. Saving files as PDF

You can save documents and text files in PDF format. The PDF file will be saved to the folder that contains the document or text file.

How to save a file in the PDF format:

  1. Select a file in the display area.

  2. Use one of the following methods:

    • Click the More actions iconin the tool bar. Click on Save as PDF.

    • Select Save as PDF from the context menu.

    • Use the More actions iconin the Viewer.

  3. In the Save as PDF window, you can enter a name for the PDF file. Click on Save.
