9.9.2. Setting up WebDAV under Windows

How to set up WebDAV access under Windows:

  1. In the navigation area of the Windows Explorer select the entry This PC.

    Click on the Computer tab at the top. Click on Add network address. The Add network address window opens. Click on Next.

  2. Select Select custom network address. Click on Next.

  3. Enter the following address in the Internet or network address input field.

    webdav://<address>/servlet/webdav.infostore

    Replace <address> with the groupware server's IP address or URL. Click on Next.

  4. Enter your username and your password for the Groupware Server. Click on OK.

  5. On the next page you can assign a name to the network address. Click on Next.

  6. Click on Finish.

Result: Below This PC you can access your Drive app files.
