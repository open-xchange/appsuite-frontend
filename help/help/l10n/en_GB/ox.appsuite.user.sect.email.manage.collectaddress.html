E-Mail

Organising E-Mail messages

5.5.4. Collecting addresses

You can collect E-Mail addresses by adding the addresses to an address book. The following options are available:

  • Automatically collect new email addresses when sending or reading email messages by activating this function in the Email settings.

  • Manually adding email addresses to an address book

How to manually add an email address to an address book:

  1. Select an E-Mail.

  2. Click on the sender's or a recipient's name in the detail view.

  3. Click on Add to address book in the pop-up.

Note: This function is only available if the contact has not been added yet.
