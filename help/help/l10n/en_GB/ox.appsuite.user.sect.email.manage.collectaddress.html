E-Mail

Managing E-Mail messages

5.7.4. collect addresses

The following options are available:

  • Automatically collect new E-Mail addresses when sending or reading E-Mail messages by activating this function in the E-Mail settings.

  • Manually adding email addresses to an address book

How to manually add an email address to an address book:

  1. Select an E-Mail.

  2. Click on the sender's or a recipient's name in the detail view.

    Click on Add to address book in the pop-up.

Note: This function is only available for new contacts.
