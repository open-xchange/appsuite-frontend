5.7.15. Cleaning up E-Mail folders

Depending on the E-Mail client's settings, E-Mail messages that you deleted from within an E-Mail client like Apple Mail, MS Outlook or Mozilla Thunderbird, might be deleted only after having signed out from the client. In the meantime, those E-Mail messages will be shown as crossed out in the groupware. To delete those E-Mail messages clean up the E-Mail folder.

Note: Depending on the configuration, this function is not available as such E-Mail messages are not displayed.

How to clean up an E-Mail folder:

  1. In the folder view, select the folder that you want to clean up.

  2. Click the Actions icon next to the folder name. Click on Clean up.
