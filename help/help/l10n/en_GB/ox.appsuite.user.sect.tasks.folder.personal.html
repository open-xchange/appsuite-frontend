Adding Task Folders

8.8.1. Adding personal task folders

You can create additional personal task folders below My tasks.

How to create a new personal task folder:

  1. In the folder view, click on Add new folder. A menu opens.

    Click on Personal folder. A window opens.

  2. Enter a name. In case the new folder should be a public folder, enable Add as public folder. Click on Add.
