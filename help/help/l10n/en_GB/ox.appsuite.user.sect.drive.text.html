9.5. Creating or Editing Text Files

The following options are available:

  • create text files with plain text

  • edit existing text files

How to create a new text file:

  1. Open a folder in the folder view.

    Note: Open a folder for which you have the appropriate permissions to create objects.

  2. Click on New in the tool bar. Click on Add note.

  3. Enter a title.

    Enter the text or paste text from the clipboard.

    In order to edit the text, use common techniques from familiar applications.

How to edit a text file:

  1. Open a folder containing text files, in the folder view.

    Note: Open a folder for which you have the permission to create objects.

  2. Use one of the following methods:

    • Select a text file. Click on Edit in the tool bar.

    • Double-click on a text file.

    • Display a text file in the Viewer. Click on Edit.

    The text will be opened for editing.

  3. In order to edit the text, use common techniques from familiar applications.
