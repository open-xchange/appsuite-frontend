8.6. Editing Tasks

Learn how to edit a task's data later.

Prerequisite: You have the permission to edit objects in the folder containing the task.

How to edit a task:

  1. Select a task from the list.

  2. Click on Edit in the tool bar. The task's data are displayed.

  3. Edit the data.

  4. Click on Save.
