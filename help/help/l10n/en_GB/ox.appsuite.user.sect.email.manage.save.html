5.5.10. Saving email messages

You can save an email as a text file. The text file will have the file extension EML.

How to save an email:

  1. Select an E-Mail.

  2. Click the More icon in the tool bar. Select Save as file from the menu.

    You can also use the Actions icon in the detail view.

  3. Follow the instructions for saving the EML file.
