9.7.1. Sending files as E-Mail attachments

You can send the current versions of files as E-Mail attachments.

How to send files as an E-Mail attachment:

  1. Select one or several files in the display area.

  2. Use one of the following methods:

    • Click the More actions icon in the tool bar. Click on Send by email.

    • Click on Send by e-mail in the context menu.

    • Use the More actions icon in the Viewer.

  3. In the E-Mail edit window, fill in the details to send a new E-Mail.
