8.2. Viewing Tasks

Learn how to display the tasks in a tasks folder in different ways:

How to display a task:

  1. Open a task folder in the folder view.

  2. Click on a task in the list. The task's data is displayed in the detail view.

Options:

  • To sort the task list or to only display due tasks, click the Sort icon above the list.

  • You can open the task in a window by double-clicking on the task in the list.
