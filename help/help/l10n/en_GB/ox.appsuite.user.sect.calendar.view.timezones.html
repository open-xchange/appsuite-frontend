7.2.2. Displaying multiple time zones

In addition to the default time zone, you can display time zones that have been marked as favourites by you. The default time zone can be changed in the basic settings.

How to display multiple time zones in a calendar view:

  1. Click on View in the tool bar. Select one of the following entries: Day, Work week or Week.

  2. Click on the Timezone button on the upper left side of the calendar sheet.

    A menu opens. Enable a time zone in the menu below Favourites.

    By clicking on Manage favourites you can add/remove time zones to/from favourites.

User interface:

  • Day, Work week, Week, Month or Year calendar view.
