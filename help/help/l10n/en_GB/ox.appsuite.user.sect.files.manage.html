Some of the techniques for managing files require having all ready set up your own folders. Information on creating folders can be found in Creating Files or Folders.

  • Sending files as a link

  • Sending files as Email attachments

  • Showing the link to a file

  • Editing file names

  • Creating or editing descriptions

  • Moving files or folders

  • Copying files

  • Adding files to the portal

  • Deleting files

  • Locking or unlocking files

  • Working with versions
