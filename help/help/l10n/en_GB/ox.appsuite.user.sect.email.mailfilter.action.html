5.8.3. Adding an action

Use an action to define what happens with an E-Mail that matches the conditions. You can set one or multiple actions.

How to add an action in the Create new rule window:

  1. Click on Add action. Select an action from the menu.

  2. Depending on the action, additional buttons and inputs fields are displayed. Use such functions to complete the action.

  3. You can add additional actions.

    To delete an action, click the Delete icon next to the action.
