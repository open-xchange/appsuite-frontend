9.2. Viewing Files

By default, the content of your personal files folder is displayed. The folder is named My files.

How to display a file:

  1. Click on View in the tool bar. Select one of the following entries: List, Icons, Squares.

  2. If the folder tree is closed, click on View in the tool bar. Enable Folder view.

    Open a folder in the folder tree.

  3. Click on a file in the display area. The data are displayed in the pop-up.

Tip: In order to quickly find a certain file, use the search function.
