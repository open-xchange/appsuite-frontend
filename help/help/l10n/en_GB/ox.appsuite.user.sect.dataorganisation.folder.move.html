11.1.7. Moving folders

You can move subfolders from your personal folders. For other folders you need the appropriate permissions.

How to move a folder:

  1. In the folder tree, select the folder that you want to move.

  2. Click the Folder-specific actions icon next to the folder name. Click on Move.

  3. Select a folder in the Move folder window. Click on Move.
