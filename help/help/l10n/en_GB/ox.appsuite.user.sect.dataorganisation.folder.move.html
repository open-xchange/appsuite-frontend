Managing Data with Folders

10.1.7. Moving folders

You can move subfolders from your personal folders. For the target folder you need permission to create subfolders.

How to move a folder:

  1. In the folder view, select the folder that you want to move.

  2. Click the Actions icon next to the folder name. Click on Move.

  3. Select a folder in the Move folder window. You can create a new folder by clicking on Create folder.

  4. Click on Move.
