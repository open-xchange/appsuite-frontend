7.1.8. The Calendar list

Displays a list of the appointments of the calendars selected in the folder view.

In order to display the list view, click on View in the tool bar. Enable List.

Content

  • Each day with appointments shows a header with the date.

  • The following details are displayed for each appointment: date, time, colour, private appointment icon, subject and location.
