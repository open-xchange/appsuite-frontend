Chapter 3. First Steps

What you should do initially.

  • Get familiar with the common user interface.

  • Get familiar with the keyboard input.

  • Customise the basic settings.

  • Check your personal contact data. Change or complete it, if required.

  • For security reasons you should change your password.

  • Define whether you want to receive a notification for new events like receiving a new E-Mail or appointment invitation.

  • In order to access your groupware data from local clients or apps, you can install clients or apps for workstations or mobile devices. Depending on the configuration, use one of the following methods:

    • manually download and install clients and apps

    • use a wizard to set up clients and apps
