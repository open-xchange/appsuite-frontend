6.1.7. The contact editing window

This window is used when creating a new contact or editing an existing one.

In order to open the window, do one of the following:

  • Select New contact from the tool bar. Click on New contact.

  • Select a contact. Click on Edit in the tool bar.

Content

  • Contact picture

    Click on the contact picture to edit it or to upload a new contact picture.

  • Contact data input fields:

    • First name, Last name

    • Company, Department

    • E-mail 1, Cell phone

  • Elements for adding additional input fields:

    • Add personal info

    • Add business info

    • Add email, phone, fax

    If clicking the Remove field icon , an additional field and its data will be deleted.

  • Note input field

  • Add postal address. Adds input fields for different postal addresses.

  • Add attachment. Adds files.
