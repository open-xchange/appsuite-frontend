6.7. Editing Contacts or Distribution Lists

Learn how to edit the data of contacts or distribution lists later.

Prerequisite: You have the appropriate permissions to edit objects in the address book containing the contact or distribution list.

How to edit a contact or distribution list:

  1. Select a contact or a distribution list from the list.

  2. Click on Edit in the tool bar. The data are displayed.

  3. Edit the data.

  4. Click on Save.
