5.8.5. Creating new rules for the subject and the sender

You can easily create a new rule for an E-Mail's subject and sender when viewing the E-Mail in the detail view.

How to create a new rule for the subject and the sender:

  1. Select an E-Mail.

  2. Click the More actions icon in the detail view. Click on Create filter rule. The Create new rule window opens.

  3. Enter a rule name. You can adjust the conditions.

    Set an action.

  4. In order to save the rule, click on Save.

    To save the rule and apply it to existing E-Mail messages, click on Save and apply. Select the folder containing the E-Mail messages to which the rule should be applied.
