9.8.2. Using cloud storages

The following options are available:

  • You can use the folder tree to access the cloud storage contents. Depending on the cloud storage content, you can use the menu bar functions, e.g. to view pictures or to copy data.

  • Depending on the content, you can use the menu bar functions, e.g. to view pictures or to copy data.
