8.8.1. Sharing tasks

You can share your tasks with internal as well as external partners. Depending on the requirements, different methods exist.

  • To make a folder available for internal users or external partners, proceed as follows:

    • Create a new personal or public tasks folder.

    • Share this folder.

    You can also share an existing folder.

  • If another internal user shared a task folder with you, you can access this folder in the folder tree.

  • To invite external partners to a task, proceed as follows:

    • When creating the task, add external partners as external participants.
