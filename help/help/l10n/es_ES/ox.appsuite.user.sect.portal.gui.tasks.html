4.1.6. Widget Tareas

Muestra las tareas sin terminar. Puede hacer lo siguiente:

  • Si pulsa en una tarea, se abrirá una ventana emergente. La ventana emergente muestra los datos de la cita.

  • Para cerrar una ventana emergente, pulse el icono Cerrar en la parte superior derecha.
