10.4.2. Inicio de una conversación con contactos

Se puede comunicar con contactos de la libreta de direcciones global iniciando una conversación con un contacto o con varios.

Cómo iniciar una conversación con contactos de la libreta de direcciones global:

  1. Abra la libreta de direcciones global en la aplicación Libreta de direcciones. Seleccione al menos un contacto.

  2. Pulse Iniciar conversación en la barra de herramientas.

    • Si seleccionó solo un contacto, puede introducir un mensaje inmediatamente.

    • Si seleccionó varios contactos, se abre una ventana para crear una conversación grupal.

      Escriba un nombre para la conversación grupal. Puede añadir y eliminar participantes si es necesario.

      Pulse en Crear conversación.

Interfaz de usuario:

  • La barra de herramientas de Libreta de direcciones
