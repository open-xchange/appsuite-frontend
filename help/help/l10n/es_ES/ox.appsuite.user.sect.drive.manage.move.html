9.6.4. Mover ficheros o carpetas de Ficheros

Puede mover los siguientes objetos a otra carpeta:

  • uno o varios ficheros

  • una o varias carpetas

  • una combinación de ficheros y carpetas

Requisito: Debe tener permisos adecuados para crear objetos en la carpeta de destino.

Cómo mover objetos a otra carpeta:

  1. Seleccione los objetos en el área de visualización.

  2. Use uno de los siguientes métodos:

    • Pulse el icono Más acciones de la barra de herramientas. Pulse Mover.

    • Seleccione Mover en el menú contextual.

    • Use arrastrar y soltar para mover los elementos a una carpeta en la vista de carpeta.

  3. Seleccione una carpeta en la ventana Mover. Puede crear una carpeta nueva pulsando en Crear carpeta.

    Pulse Mover.

Interfaz de usuario:

  • El área de visualización de Ficheros
