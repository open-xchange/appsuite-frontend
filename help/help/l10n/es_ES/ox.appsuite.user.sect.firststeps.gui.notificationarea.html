2.4.8. El área de notificación

Nota: Lo siguiente es una representación esquemática.

Muestra notificaciones sobre los siguientes eventos:

  • nuevas invitaciones a citas

  • recordatorio de una cita o tarea programadas

Las siguientes acciones abren el área de notificación:

  • Pulsar el icono Notificación de la barra de menús. El icono se muestra al recibir una nueva notificación.

  • Dependiendo de la configuración, el área de notificación se abre automáticamente si hay una nueva notificación.

Contenido

  • Notificaciones sobre nuevas invitaciones y eventos próximos

  • Controles para editar notificaciones: aceptar, rechazar, recordar otra vez, borrar
