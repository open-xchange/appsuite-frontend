9.2.2. Mantener una presentación

Puede mantener las presentaciones en el formato habitual de Office.

Cómo mantener una presentación:

  1. Abra una carpeta que contenga presentaciones.

  2. Abra la presentación en el Visor. Pulse el icono Presentar . La presentación se abrirá en una página nueva.

    Para navegar por las páginas, utilice los elementos de navegación que hay junto a la hoja o las vistas previas que se encuentran bajo la hoja. También puede utilizar las teclas del cursor.

    Si pulsa el icono Mostrar participantes se muestran los participantes de la presentación en la barra lateral.

  3. Puede invitar a usuarios externos a la presentación fuera del sitio web. Para ello envíe la url de la presentación seleccionada a dichos usuarios.

  4. Pulse en Empezar presentación en la barra barra de herramientas.

    Durante la presentación, están disponibles las siguientes funciones.

    • Para mostrar una barra de funciones, pase el ratón sobre la parte inferior de la pantalla. Las siguientes funciones están disponibles: navegar, visualizar una hoja concreta, visualizar la lista de participantes, detener la presentación, activar el modo de pantalla completa.

    • Si pulsa en Detener presentación, la presentación se detiene en la hoja actual.

    • Si pulsa el icono Cambiar a pantalla completa, se activa el modo pantalla completa. Puede utilizar la tecla [Esc] o la barra de funciones de la parte inferior de la pantalla para abandonar el modo pantalla completa.

  5. Para finalizar la presentación, pulse en Finalizar presentación en la barra de herramientas.
