8.8.1. Adición de carpetas de tareas personales

Puede crear carpetas de tareas personales adicionales debajo de Mis tareas.

Cómo crear una nueva carpeta de tareas personales:

  1. En la vista de carpeta, pulse en Añadir nueva carpeta. Se abrirá un menú.

    Pulse en Carpeta personal. Se abrirá una ventana.

  2. Introduzca un nombre. En caso de que la nueva carpeta deba ser una carpeta pública, marque Añadir como carpeta pública. Pulse en Añadir.
