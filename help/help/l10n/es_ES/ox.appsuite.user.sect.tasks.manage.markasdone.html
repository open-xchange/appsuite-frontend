8.6.2. Marcado de tareas como realizadas

Puede marcar una tarea o varias de una vez como realizadas.

Cómo marcar una tarea como realizada:

Requisito: Debe tener permiso para crear objetos en la carpeta que contiene la tarea.

  1. Seleccione una tarea de la lista.

  2. Seleccione Marcar como completada en la barra de herramientas.
