Ficheros

9.1. Los componentes de Ficheros

Se puede encontrar una introducción a la interfaz de usuario en El interfaz de usuario.

Árbol de carpetas Muestra las carpetas. Le permite navegar por la estructura de carpetas. Para abrir el árbol de carpetas, pulse el botón Vista de la barra de herramientas. Marque Vista de carpeta. Están disponibles las siguientes funciones:

  • Si ha pulsado en una carpeta, se muestran sus ficheros.

  • El icono Acciones específicas de carpeta que hay junto a la carpeta seleccionada nos ofrece funciones para organizar datos y para intercambiar datos.

Barra de herramientas Contiene los siguientes botones:

  • Botón Nuevo. Contiene las siguientes funciones:

    • Subir nuevo fichero. Crea un nuevo fichero.

    • Añadir nota. Crea un fichero de texto sencillo.

    • Nuevo documento de texto. Crea un nuevo documento de texto.

    • Nueva hoja de cálculoCrea una nueva hoja de cálculo.

  • Si ha abierto una carpeta personal, se muestra el botón Compartir carpeta. Si se pulsa este botón, se creará una publicación de dicha carpeta.

  • Si la carpeta contiene imágenes, se mostrarán iconos para reproducir ficheros multimedia.

  • Si se ha seleccionado un fichero, se mostrará el icono Descargar.

  • Si se ha seleccionado un fichero, se mostrará el icono Borrar.

  • Si se ha seleccionado un fichero, se mostrará el icono Más . Si se pulsa en el icono, se abre un menú con los siguientes elementos: Enviar por correo, Enviar como enlace interno, Mostrar enlace interno, Añadir al portal, Mover, Copiar, Bloquear.

    Nota: Si se han seleccionado varios ficheros, algunas funciones no están disponibles.

  • Botón Vista. Seleccione una de las siguientes vistas para la vista de detalle:

    • Lista

    • Iconos

    • Panel

Nota: Algunos botones sólo se muestran si la carpeta seleccionada contiene ficheros.

Área de visualización en la vista Lista Por cada fichero se muestra una fila con el siguiente contenido.

  • Una casilla de verificación para seleccionar uno o varios ficheros.

  • Un icono para cada fichero de la carpeta seleccionada. Se muestra una vista previa del fichero si es posible.

  • Nombre y tamaño del fichero

  • Fecha y hora del último cambio

Si se pulsa sobre un fichero, se abre una ventana emergente. Se muestran más datos y funciones.

Área de visualización en la vista Iconos Ruta de navegación a la carpeta seleccionada.

  • Un icono por cada fichero de la carpeta seleccionada. Se muestra una vista previa del fichero, si es posible.

  • El nombre del fichero

Si se pulsa en un fichero, se abre una ventana emergente. Muestra más datos y funciones.

Área de visualización en la vista Panel Muestra un panel por cada fichero de la carpeta seleccionada. Si está disponible, se muestra una previsualización. Si se pulsa sobre un fichero, se abre una ventana emergente que incluye más datos y funciones.

Ventana emergente Si se pulsa en un fichero, se muestra una ventana emergente. Muestra la siguiente información:

  • el botón Abrir

    Si se pulsa en un documento, se muestra el botón Ver en lugar del botón Abrir.

  • Si se pulsa sobre un fichero editable, se muestra el botón Editar.

  • Botones Descargar, Borrar

  • Si se pulsa el icono Más se abre un menú con los siguientes elementos: Compartir este fichero, Enviar por correo, Enviar como enlace interno, Mostrar enlace interno, Añadir al portal, Renombrar, Editar descripción, Mover, Copiar, Bloquear.

    Nota: Algunos botones podrían no mostrarse si no se tienen los permisos adecuados.

  • Si se pulsa en un documento, el botón Editar como nuevo se muestra en el submenú Más.

  • nombre del fichero

  • una vista previa del fichero, si está disponible

  • comentarios del fichero

  • Ruta de navegación a la carpeta seleccionada. Para abrir otra carpeta, pulse en la ruta.

  • Botones para subir una nueva versión.

  • El botón Mostrar historial de versiones. Si se pulsa sobre este botón, se muestra una lista de versiones. La versión actual se muestra la primera. Para cada versión se muestra la siguiente información:

    • el nombre de fichero de la versión

    • comentarios de la versión, si están disponibles

    • nombre del usuario que subió la versión

    • fecha y hora de la carga de la versión
