Correo electrónico

Los componentes de Correo electrónico

5.1.7. El elemento emergente de Correo electrónico

Muestra los datos de contacto del remitente o destinatario que haya seleccionado en la vista de detalle.

Contenido

  • Botones para funciones usadas frecuentemente

    • Si el contacto aún no está guardado en una libreta de direcciones, se mostrará Añadir a libreta de direcciones. Si se pulsa el botón, se abre la ventana para crear un nuevo contacto.

    • Enviar correo. Abre la ventana para enviar un nuevo correo al contacto.

    • Invitar. Abre la ventana para crear una cita con este contacto.

  • Icono Más acciones . Si se pulsa el icono, se abrirá un menú con funciones usadas frecuentemente para organizar contactos.

  • La Vista halo del contacto.
