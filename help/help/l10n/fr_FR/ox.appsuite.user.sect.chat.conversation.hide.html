10.2.5. Fermer une discussion instantanée

Pour garder une vue synthétique de la liste des discussions, vous pouvez fermer ou rouvrir certaines discussions.

Pour fermer une discussion instantanée :

  1. Dans la liste des discussions, sélectionnez une discussion privée, une discussion de groupe ou un salon public.

  2. Cliquez sur l'icône Actions supplémentaires en haut à droite.

    Sélectionnez Fermer la discussion.

La discussion est fermée dans la liste et tous les messages associés disparaissent du champ de discussion instantanée. Des icônes permettant de créer une nouvelle discussion apparaîtront dans le champ de discussion instantanée.

Pour ouvrir une discussion fermée :

  1. Cliquez sur Historique dans la liste des discussions. Toutes les discussions apparaissent alors dans le champ de discussion instantanée.

  2. Cliquez sur Ouvrir à côté d'une discussion.
