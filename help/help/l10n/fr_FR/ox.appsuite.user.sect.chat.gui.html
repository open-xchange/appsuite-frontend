Discussion

10.1. La fenêtre Discussion

Cette fenêtre est utilisées pour toute les fonctions de Discussion :

  • recherche et affichage de contacts, de groupes et de salons publics

  • création et modification de groupes et de salons publics

  • échange de messages directs avec un contact individuel, échange de messages au sein d'un groupe ou d'un salon public

Pour ouvrir la fenêtre, cliquez sur l'icône Discussion instantanée dans la barre de menu. Si vous avez des messages non lus, l'icône présente un point rouge. Vous pouvez également utiliser le lanceur d'applications dans la barre de menu.

Contenu

  • Dans la barre de titre : icône Ancrer à droite . Ancre la fenêtre de discussion au bord droit de l'écran.

    Pour ajuster la largeur de la fenêtre ancrée, faites glisser vers la gauche ou vers la droite la bordure séparant la fenêtre de la vue détaillée.

    Pour pouvoir déplacer librement la fenêtre, cliquez sur l'icône Détacher la fenêtre figurant sur la droite. Le texte qui suit se réfère à la fenêtre déplaçable.

  • Votre image de profil, votre nom d'affichage, votre état de présence

  • Icône Nouveau . Ouvre un menu comportant des fonctions supplémentaires :

    • créer une discussion privée avec un contact unique

    • créer une discussion de groupe avec plusieurs participants

    • créer un salon public que d'autres contacts peuvent rejoindre

  • Champ de saisie Rechercher ou démarrer une discussion instantanée. La saisie d'un terme à rechercher fait apparaître les éléments suivants dans la liste des discussions :

    • Contacts, groupes et salons publics dont le nom correspond au terme recherché

      Un clic sur un contact démarre une discussion privée avec ce contact

    • Discussions contenant le terme recherché

  • Discussions instantanées. Contient la liste des discussions ouvertes. Via les réglages des discussions, vous pouvez définir si les discussions doivent être regroupées en Discussions privées, Discussions de groupe et Salons.

    • Pour les discussions privées, l'état de présence du contact est affiché.

    • Lorsque vous cliquez sur une discussion instantanée, les messages envoyés et reçus dans le cadre de cette discussion s'affichent.

    • La vue en liste des discussions peut être ajustée dans les réglages des discussions instantanées.

  • Sous la liste des discussions instantanées, vous trouvez des fonctions permettant d'afficher des éléments spécifiques dans le champ de discussion :

    • Tous les salons. Affiche tous les salons publics dans le champ de discussion instantanée. Vous pouvez rejoindre un salon en cliquant sur Rejoindre.

    • Tous les fichiers. Affiche dans le champ de discussion instantanée tous les fichiers ayant été utilisés dans le cadre des discussions.

    • Historique. Affiche toutes les discussions instantanées ouvertes ou fermées dans le champ de discussion instantanée.

  • Le titre du champ de discussion instantanée

    • Affiche le nom du contact, du groupe ou du salon sélectionné dans la liste des discussions.

      Pour les discussions privées, l'état de présence du contact est affiché.

      Pour les groupes ou les salons, les noms des membres sont également affichés.

      L'icône Actions supplémentaires ouvre un menu proposant des fonctions supplémentaires :

      • Fermer la discussion instantanée. Ferme la discussion instantanée actuelle et les messages associés. Pour afficher de nouveau cette discussion, cliquez sur Historique dans la liste.

      • Pour les discussions de groupe ou pour un salon :

        Modifier la discussion ouvre une fenêtre permettant de modifier la discussion.

        Vous pouvez quitter la discussion en cliquant sur Quitter la discussion instantanée.

    • Affiche le nom de la fonction que vous avez sélectionnée sous le champ de discussion instantanée.

  • Champ de discussion instantanée

    • Contient les messages envoyés et reçus dans le cadre de la discussion sélectionnée dans la liste.

      Pour les discussions privées, une icône indique l'état de lecture du message :

      • Une coche grise : le message est arrivé sur le serveur.

      • Deux coches grises : le message a été transféré au destinataire par le serveur.

      • Deux coches bleues : le message a été lu par le destinataire.

    • Affiche les notifications système si des discussions de groupe ou des salons publics sont créés ou modifiés.

    • Affiche le résultat de la fonction que vous avez sélectionnée sous le champ de discussion instantanée.

  • Éléments permettant de rédiger un message

    • Icône Transférer un fichier . Ajoute des fichiers à la discussion

    • Champ de message. Celui-ci vous permet de saisir un message à envoyer dans le cadre de cette discussion instantanée.

    • Icône Ajouter un émoji . Ouvre ou ferme la fenêtre d'ajout d'émojis.

    • Icône Envoyer . Envoie le message.
