Afficher les rendez-vous

7.2.3. Comment les rendez-vous sont-ils affichés ?

Dans la vue d'agenda, les rendez-vous sont affichés avec les couleurs suivantes en fonction de la disponibilité :

  • Provisoire : jaune

  • Disponible : vert

  • Planifié : bleu

  • Absent : rouge

Les rendez-vous privés sont toujours affichés en gris.
