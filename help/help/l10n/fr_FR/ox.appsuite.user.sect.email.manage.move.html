Organisation des courriels

5.5.1. Déplacer des messages

Vous pouvez déplacer un message isolé ou un fil de discussion complet dans un autre dossier.

Pour déplacer un message :

Prérequis : vous devez disposer du droit de créer des objets dans le dossier cible.

  1. Sélectionnez un courriel.

  2. Cliquez sur l'icône Actions dans la barre d'outils. Sélectionnez Déplacer depuis le menu.

    Vous pouvez aussi utiliser l'icône Actions dans la vue détaillée.

  3. Sélectionnez un dossier dans la fenêtre Déplacer. Cliquez sur le bouton Déplacer.

Conseil : vous pouvez également déplacer plusieurs courriels à la fois.

Conseil : pour déplacer des courriels par glisser-déposer, sélectionnez un ou plusieurs messages dans la barre latérale. Faites glisser les messages sélectionnés vers un dossier de l'arborescence des dossiers.

Conseil : pour déplacer tous les courriels d'un dossier, sélectionnez au moins deux courriels dans la liste. Cliquez ensuite sur Déplacer tous les messages vers un autre dossier dans la vue détaillée.
