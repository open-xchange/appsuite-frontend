Carnet d'adresses

6.2. Afficher des contacts

Par défaut, les contacts du Carnet d'adresses global sont affichés. Votre carnet d'adresses personnel se trouve dans l'arborescence des dossiers sous Contacts.

Pour afficher un contact :

  1. Ouvrez un carnet d'adresses dans l'arborescence des dossiers.

  2. Il existe plusieurs méthodes pour trouver rapidement le contact désiré :

    • Pour n'afficher que les contacts ayant une propriété spécifique, utilisez la fonctionnalité de recherche.

    • Pour afficher les contacts avec une initiale donnée, cliquez sur la lettre dans la barre de navigation.

  3. Cliquez sur un contact dans la liste. Les données du contact sont affichées dans la vue détaillée.

  4. Il y a plusieurs méthodes pour afficher un autre contact :

    • Cliquez sur un autre contact dans la liste.

    • Utiliser les flèches pour naviguer dans la liste.
