3.6. Configurer des comptes de réseaux sociaux

Pour configurer un compte permettant d'accéder à des réseaux sociaux

  1. Cliquez sur l'icône Menu système à droite de la barre de menu. Cliquez sur l'élément de menu Réglages.

  2. Cliquez sur Comptes de courrier et sur les réseaux sociaux dans la barre latérale.

  3. Cliquez sur Ajouter dans la zone d'affichage. Choisissez un réseau social dans le menu. Vous serez redirigé vers la page du réseau social où vous devrez vous authentifier.

    Remarque : Le menu affiche uniquement les réseaux sociaux pour lesquels vous n'avez pas encore configuré de compte dans le collecticiel.

  4. Saisissez vos éléments d'authentification auprès du réseau social. Pour terminer la procédure, cliquez sur le bouton adéquat.

Remarque : vous pouvez également utiliser les Composants de portail pour définir un compte d'accès aux réseaux sociaux.
