Premiers pas avec le logiciel de travail collaboratif

2.2. Mode d'emploi

Les techniques suivantes doivent être connues :

  • Utiliser le bouton gauche de la souris pour cliquer sur des éléments

  • Utiliser le glisser-déposer

  • Sélectionner plusieurs éléments en appuyant sur la touche [Ctrl] ou [Cmd] sur un Macintosh

  • Travailler avec plusieurs fenêtres

  • Ouvrir, enregistrer et fermer des fichiers

  • Ouvrir, fermer et sélectionner des fichiers dans une arborescence de dossiers

Des informations sur ces techniques se trouvent, par exemple, dans la documentation de votre système d'exploitation.
