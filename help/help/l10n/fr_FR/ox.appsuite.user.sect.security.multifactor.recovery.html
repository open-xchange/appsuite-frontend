11.7.5. Récupération de la vérification

Vous aurez besoin d'une solution de récupération, par exemple, si vous perdez l'appareil utilisé pour la vérification. Dans ce cas, vous pourrez alors vous connecter au collecticiel grâce à cette méthode de récupération.

Pour vous connecter au moyen d'une méthode de récupération :

  1. Récupération à l'aide d'un code de récupération :

    1. Cliquez sur J'ai perdu mon appareil dans la fenêtre Vérification en deux étapes ou Sélectionner la méthode de vérification en deux étapes.

      Une fenêtre vous invite à saisir le code de récupération.

    2. Saisissez le code de récupération.

      Si, lorsque vous avez configuré la méthode de récupération, vous avez téléchargé le code de récupération sous forme de fichier texte, vous pouvez également cliquer sur Transférer un fichier de récupération. Sélectionnez le fichier texte contenant le code de récupération.

    3. Peu de temps après, vous devriez recevoir un SMS comportant un code d'authentification à 4 chiffres. Saisissez ce code.

  2. Récupération à l'aide d'un code envoyé par SMS :

    1. Cliquez sur J'ai perdu mon appareil dans la fenêtre Vérification en deux étapes ou Sélectionner la méthode de vérification en deux étapes.

      Une fenêtre vous invite à saisir le code de récupération.

    2. Peu de temps après, vous devriez recevoir un SMS comportant un code d'authentification à 4 chiffres. Saisissez ce code.
