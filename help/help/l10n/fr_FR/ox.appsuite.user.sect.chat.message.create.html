Travailler avec des messages

10.3.1. Lire des messages, en envoyer et y répondre

Vous disposez des options suivantes :

  • Rechercher des messages au sein d'une discussion et les lire. Vous êtes notifié de l'arrivée de nouveaux messages des manières suivantes :

    • Si vous avez des messages non lus, l'icône de la discussion dans la barre de menu présente un point rouge.

    • Vous pouvez définir les notifications suivantes dans les réglages des discussions instantanées :

      • Un son est joué à l'arrivée d'un message.

      • Si le collecticiel est en arrière-plan, une notification de bureau est affichée.

      • Si vous êtes déconnecté, vous recevez un courriel de notification incluant le contenu du message de discussion instantanée. Si vous répondez à ce courriel, la réponse est insérée dans la discussion instantanée.

  • envoyer des messages et des fichiers à un groupe ou à un contact individuel

  • Répondre à un message antérieur. Le message d'origine fera l'objet d'une citation dans votre réponse.

Pour lire un message :

  1. Sélectionnez une discussion dans la liste.

    Les discussions comportant des messages non lus seront affichées comme suit :

    • Le nom de la discussion apparaît en caractères gras.

    • Un nombre indique le nombre de messages non lus.

  2. Lisez les messages de la conversation. Pour consulter les messages antérieurs, utilisez les icônes sur le côté droit afin de remonter dans la discussion.

Options :

  • Pour rechercher des discussions, saisissez un terme à rechercher dans le champ de saisie Rechercher ou démarrer une discussion instantanée

Pour envoyer un message :

  1. Utilisez l'une des méthodes suivantes :

    • Pour démarrer une nouvelle discussion instantanée, cliquez sur l'icôneNouveau .

      Dans le menu, sélectionnez un type de conversation.

    • Pour utiliser une discussion existante, sélectionnez-en une dans la liste. Les messages échangés jusqu'à présent dans le cadre de cette discussion seront alors affichés.

  2. Saisissez un message dans le champ de saisie.

    Si vous fermez la fenêtre de discussion ou que vous ouvrez une autre discussion en cours de saisie d'un message, cette saisie sera préservée tant que vous resté connecté au collecticiel.

  3. Pour envoyer le message, appuyez sur la touche [Entrée] ou cliquez sur l'icône Envoyer .

    Pour les discussions privées, une icône figurant à côté du message indique son état de lecture :

    • Une coche grise : le message est arrivé sur le serveur.

    • Deux coches grises : le message a été transféré au destinataire par le serveur.

    • Deux coches bleues : le message a été lu par le destinataire.

Options :

  • Pour insérer un saut de ligne à l'intérieur d'un message, appuyez sur la combinaison de touches [Maj]+[Entrée].

  • Si vous envoyez un message à un groupe, vous pouvez vous adresser à un contact spécifique dans ce groupe. Pour cela, saisissez le caractère [@] suivi du nom de la personne, puis du message. Le contact visé verra une icône rouge signalant le nom du groupe dans la liste.

  • Pour envoyer des émojis, cliquez sur l'icône Ajouter un émoji à côté du champ de saisie. Une fenêtre d'émojis s'ouvre alors. Sélectionnez au moins un émoji.

    Pour fermer la fenêtre, cliquez à nouveau sur l'icône Ajouter un émoji .

  • Pour envoyer un fichier, cliquez sur l'icône Transférer un fichier à côté du champ de saisie. Sélectionnez un ou plusieurs fichiers.

    Vous pouvez aussi faire glisser un ou plusieurs fichier(s) depuis un navigateur de fichiers ou le bureau vers le champ de discussion instantanée.

    Remarque : pour des raisons de sécurité, il n'est pas possible d'envoyer des fichiers exécutables.

  • Pour consulter tous les fichiers précédemment envoyés ou reçus, cliquez sur Tous les fichiers dans la liste.

  • Pour mettre en forme un message texte, utilisez les caractères suivants :

    • Ce mot apparaîtra en _italique_.

    • Ce mot apparaîtra en *gras*.

    • Ce mot sera ~barré~.

    • Ceci indique du `code source` au sein d'une ligne.

    • Ceci représente un ```bloc de code

      Ligne de code 1

      Ligne de code 2

      ```

      Remarque : pour insérer un saut de ligne à l'intérieur d'un extrait de code, appuyez sur la combinaison de touches [Maj]+[Entrée].

Pour répondre à un message :

  1. Dans le champ de discussion instantanée, accédez à un message.

    Cliquez sur l'icône Actions sur les messages à côté du message. Sélectionnez Répondre.

    Le message ainsi que son expéditeur seront cités dans le champ de saisie d'un message.

  2. Saisissez votre réponse dans le champ de saisie d'un message.

    Pour envoyer la réponse, appuyez sur la touche [Entrée] ou cliquez sur l'icône Envoyer .
