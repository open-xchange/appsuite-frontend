Pour lancer l'application Discussion :

  • Cliquez sur l'icône Toutes les applications dans la barre de menu. Dans le lanceur d'applications, cliquez sur Discussion.

    Vous pouvez également cliquer sur l'icône Discussion instantanée dans la barre de menu.

    La fenêtre Discussion s'ouvre alors.
