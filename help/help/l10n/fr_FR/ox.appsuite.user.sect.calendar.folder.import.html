7.7.3. Importer des agendas depuis des fichiers

Vous pouvez importer des agendas que vous avez créés dans d'autres applications. L'agenda que vous voulez importer doit être disponible sous forme de fichier iCal.

Pour importer des agendas depuis un fichier :

  1. Dans la vue des dossiers, cliquez sur Ajouter un agenda. Un menu s'ouvre alors.

    Choisissez Transférer un fichier. La fenêtre Importer depuis un fichiers'ouvre alors.

  2. Cliquez sur Transférer un fichier. Sélectionnez un fichier.

  3. Saisissez un nom pour le nouvel agenda. Cliquez sur Importer.
