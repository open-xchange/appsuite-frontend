4.2.2. Suppression des composants du portail

Vous pouvez retirer un composant du portail en le fermant.

Comment retirer un composant :

  • Cliquez sur l'icône Fermer dans le composant.

    Utilisez les réglages du portail pour afficher le composant à nouveau.
