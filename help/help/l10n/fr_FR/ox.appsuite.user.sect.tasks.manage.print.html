8.6.6. Imprimer des tâches

Pour imprimer des tâches, vous pouvez :

  • imprimer les détails d'une tâche

Pour imprimer les détails d'une tâche :

  1. Choisissez une tâche dans la liste.

  2. Cliquez sur l'icône Plus dans la barre d'outils. Dans le menu, cliquez sur Imprimer. Une fenêtre s'ouvre alors et présente un aperçu avant impression.

  3. Si nécessaire, changez les réglages de l'imprimante. Cliquez sur le bouton Imprimer.

  4. Fermez la fenêtre d'aperçu d'impression.

Conseil : vous pouvez aussi imprimer les détails de plusieurs tâches à la fois.
