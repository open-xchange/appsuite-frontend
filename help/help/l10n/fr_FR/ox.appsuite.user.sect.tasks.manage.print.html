8.9.7. Imprimer des tâches

Vous pouvez imprimer les données d'une ou plusieurs tâches à la fois.

Pour imprimer les données d'une tâche :

  1. Choisissez une tâche ou plusieurs tâches dans la liste.

  2. Cliquez sur l'icône Actions supplémentaires dans la barre d'outils. Cliquez sur Imprimer. Une fenêtre s'ouvre alors et présente un aperçu avant impression.

  3. Si nécessaire, changez les réglages de l'imprimante. Cliquez sur le bouton Imprimer.

  4. Fermez la fenêtre d'aperçu d'impression.
