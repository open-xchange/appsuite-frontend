7.9.6. Déplacer des rendez-vous vers un autre agenda

Vous pouvez déplacer des rendez-vous vers un autre agenda.

Pré-requis : vous devez avoir le droit de créer des objets dans l'agenda cible.

Pour déplacer des rendez-vous vers un autre agenda :

  1. Suivant la vue courante, utilisez l'une des méthodes suivantes :

    • Cliquez sur un rendez-vous dans la vue en calendrier. Dans la fenêtre, qui s'ouvre, cliquez sur l'icône Actions supplémentaires .

    • Sélectionnez un ou plusieurs rendez-vous dans la vue en liste. Cliquez sur l'icône Actions supplémentaires dans la barre d'outils.

  2. Cliquez sur Déplacer.

    Choisissez un agenda dans la fenêtre Déplacer. Vous pouvez créer un nouvel agenda en cliquant sur Créer un dossier.

  3. Cliquez sur Déplacer.
