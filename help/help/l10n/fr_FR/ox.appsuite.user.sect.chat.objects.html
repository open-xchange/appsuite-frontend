10.4. Créer des discussions instantanées depuis d'autres applications

Vous pouvez créer une discussion instantanée lorsque vous travaillez avec des objets du collecticiel dans d'autres applications. Vous disposez des options suivantes :

  • Répondre à un courriel à l'aide d'un message de discussion instantanée

  • Démarrer une discussion instantanée avec des contacts

  • Démarrer une discussion instantanée avec les participants à un rendez-vous

Interface utilisateur :

  • La fenêtre Discussion
