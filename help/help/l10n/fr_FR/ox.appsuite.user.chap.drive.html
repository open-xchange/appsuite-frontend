Pour démarrer l'app Fichiers

  • Cliquez sur l'icône du Lanceur d'applications dans la barre de menu. Cliquez sur Fichiers dans le Lanceur d'applications.

    Suivant la configuration, vous pouvez également cliquer sur l'icône de démarrage rapide de la barre de menu pour lancer l'application Fichiers.
