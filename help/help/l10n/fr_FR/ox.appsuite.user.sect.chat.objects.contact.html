10.4.2. Démarrer une discussion instantanée avec des contacts

Vous pouvez communiquer avec des contacts du carnet d'adresses global en démarrant une discussion instantanée avec un ou plusieurs contacts.

Pour démarrer une discussion instantanée avec des contacts du carnet d'adresses global :

  1. Ouvrez le carnet d'adresses global dans l'application Carnet d'adresses. Sélectionnez au moins un contact.

  2. Dans la barre d'outils, cliquez sur Démarrer une discussion instantanée

    • Si vous aviez sélectionné un contact uniquement, vous pouvez immédiatement saisir un message.

    • Si vous aviez sélectionné plusieurs contacts, une fenêtre de création de discussion de groupe s'ouvre alors.

      Saisissez un nom pour la discussion de groupe. Vous pouvez au besoin ajouter ou retirer des participants.

      Cliquez sur Créer une discussion.

Interface utilisateur :

  • La barre d'outils Carnet d'adresses
