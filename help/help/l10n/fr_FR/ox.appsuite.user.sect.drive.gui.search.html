9.1.1. La barre de recherche Fichiers

Permet de chercher des fichiers.

La barre de recherche est affichée si la vue des dossiers est visible. Pour afficher la vue des dossiers, cliquez sur Affichage dans la barre d'outils. Cliquez Vue des dossiers.
