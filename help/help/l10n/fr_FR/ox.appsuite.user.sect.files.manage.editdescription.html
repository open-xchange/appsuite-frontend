9.5.5. Modifier des descriptions

Vous pouvez modifier la description d'un fichier.

Pour modifier la description d'un fichier :

Pré-requis : vous devez avoir les droits permettant de modifier les objets dans le dossier contenant le fichier.

  1. Cliquez sur un fichier dans la vue détaillée. Dans la fenêtre, cliquez sur l'icône Plus . Cliquez sur Modifier la description dans le menu.

  2. Modifiez la description.

  3. Cliquez sur Enregistrer.
