Chiffrer les données avec Sécurité

13.3. Chiffrer les conversations par courriel à l'aide d'Autocrypt

La norme Autocrypt a été développée afin de fournir un chiffrement automatique des courriels. Le système fonctionne ainsi :

  • Chaque courriel envoyé inclut la clé publique de l'expéditeur dans l'en-tête du courriel. Cette clé est importée automatiquement ou manuellement par le destinataire.

Autocrypt présente les limitations suivantes :

  • Aucune vérification n'est appliquée aux clés importées automatiquement pour vérifier si elles sont correctes ou s'il s'agit de faux.

  • Pour cette raison, une clé importée automatiquement ne sera pas utilisée pour vérifier la signature de l'expéditeur.

Pour gérer les clés Autocrypt et les valider ultérieurement, vous disposez des options suivantes :

  • Importer des clés Autocrypt

  • Gérer les clés Autocrypt

  • Transférer des clés Autocrypt vers d'autres clients
