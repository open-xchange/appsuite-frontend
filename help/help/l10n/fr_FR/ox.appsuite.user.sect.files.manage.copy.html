9.5.7. Copier des fichiers

Vous pouvez déplacer un fichier ou plusieurs fichiers à la fois vers un autre dossier.

Pour copier un fichier :

Pré-requis : vous devez avoir les droits permettant de créer des objets dans le dossier cible.

  1. Cliquez sur un fichier dans la vue détaillée. Dans la fenêtre, cliquez sur l'icône Plus . Sélectionnez Copier depuis le menu.

    Vous pouvez aussi sélectionner un ou plusieurs fichiers. Cliquez sur l'icône Plus icon dans la barre d'outils. Sélectionner Copier depuis le menu.

  2. Choisissez un fichier dans la fenêtre Copier. Cliquez sur Copier.
