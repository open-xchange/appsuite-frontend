Pour lancer l'application Agenda

  • Cliquez sur l'icône Toutes les applications dans la barre de menu. Dans le lanceur d'applications, cliquez sur Agenda.

    Suivant la configuration, vous pouvez aussi cliquer sur l'icône de lancement rapide pour démarrer l'application Agenda dans la barre de menu.
