WebDAV によるファイルへのアクセス

9.8.1. Linux 環境での WebDAV のセットアップ

Linux 環境で WebDAV アクセスをセットアップする：

  1. KDE Konqueror または同等のブラウザを起動します。

  2. アドレス バーに次のアドレスを入力します。

    webdav://<address>/servlet/webdav.infostore

    <address>をグループウェアサーバのIPアドレスかURLに置き換えます。

  3. グループウェア サーバにアクセスするためのユーザ名とパスワードを入力します。

結果：ドライブアプリケーションをブラウザで開きます。
