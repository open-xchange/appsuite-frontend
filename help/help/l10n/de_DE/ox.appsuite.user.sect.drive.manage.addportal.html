9.7.7. Datei zum Portal hinzufügen

Sie können eine Datei als Widget zum Portal hinzufügen.

So fügen Sie eine Datei zum Portal hinzu:

  • Wählen Sie im Anzeigebereich eine Datei. Klicken Sie in der Werkzeugleiste auf das Symbol Weitere Aktionen . Klicken Sie auf Zum Portal hinzufügen.

  • Der Drive-Anzeigebereich
