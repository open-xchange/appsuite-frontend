7.2.3. Termine in der Listenansicht anzeigen

So zeigen Sie einen Termin in der Listenansicht an:

  1. Klicken Sie in der Werkzeugleiste auf Ansicht. Wählen Sie Liste.

  2. Öffnen Sie im Ordnerbaum einen Kalenderordner.

    Wenn Sie alle Ihre Termine aus allen Kalenderordnern sehen möchten, öffnen Sie den Ordner Alle meine Termine.

  3. Klicken Sie in der Liste auf einen Termin. Die Daten des Termins werden in der Detailansicht angezeigt.

    Sie können den Termin auf einer separaten Seite öffnen, indem Sie den Termin in der Liste doppelklicken.

  4. Um einen anderen Termin anzuzeigen, verwenden Sie eine der folgenden Methoden:

    • Klicken Sie in der Liste auf einen anderen Termin.

    • Blättern Sie mit Hilfe der Cursortasten durch die Liste.

Tipp: Sie können auch nach Terminen suchen.

Ähnliche Aktionen

  • Termine in einer Kalenderansicht anzeigen, Termin-Anlage ansehen oder speichern, Mehrere Zeitzonen anzeigen
