Datensicherheit

11.1. Automatisch abmelden

Sie können festlegen, ob Sie automatisch abgemeldet werden, wenn Sie für eine bestimmte Zeit nicht mit der Groupware gearbeitet haben.

So bestimmen Sie, ob Sie automatisch abgemeldet werden:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Einstellungen . Klicken Sie auf Einstellungen.

  2. Klicken Sie in der Seitenleiste auf Sicherheit.

  3. Wählen Sie im Anzeigebereich unter Automatisch abmelden einen Eintrag.

Verwandte Themen:

  • Extern verlinkte Bilder in E-Mails

    Authentizität von E-Mails anzeigen

    Zulassungsliste verwenden

    Aktive Clients anzeigen oder abmelden

    Zusätzliche Passwörter für Applikationen verwenden
