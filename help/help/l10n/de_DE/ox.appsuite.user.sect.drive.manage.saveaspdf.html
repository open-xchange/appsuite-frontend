9.7.6. Datei als PDF speichern

Sie können Dokumente und Textdateien im PDF-Format speichern. Die PDF-Datei wird in dem gleichen Ordner gespeichert, in dem sich das Dokument oder die Textdatei befindet.

So speichern Sie eine Datei im PDF-Format:

  1. Wählen Sie im Anzeigebereich eine Datei.

  2. Verwenden Sie eine der folgenden Methoden:

    • Klicken Sie in der Werkzeugleiste auf das Symbol Weitere Aktionen . Klicken Sie auf Als PDF speichern.

    • Wählen Sie im Kontextmenü Als PDF speichern.

    • Verwenden Sie im Viewer das Symbol Weitere Aktionen .

  3. Im Fenster Als PDF speichern bestimmen Sie den Namen der PDF-Datei. Klicken Sie auf Speichern.
