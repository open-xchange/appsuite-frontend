Freigaben

9.6.1. Daten freigeben

Sie haben die folgenden Möglichkeiten:

  • Eine Datei oder mehrere Dateien zum Lesen oder Bearbeiten freigeben

  • .. und das.

So geben Sie Dateien zum Lesen oder Bearbeiten frei:

  1. In der App Drive wählen Sie einen Ordner.

    Wählen Sie im Anzeigebereich eine Datei oder mehrere Dateien.

  2. Klicken Sie in der Werkzeugleiste auf Teilen. Das Fenster Freigeben öffnet sich

  3. Um zu bestimmen, für wen Sie die Daten freigeben wollen, verwenden Sie eine dieser Methoden:

    • Um andere Benutzer oder externe Partner mit Hilfe ihrer E-Mail-Adresse einzuladen, wählen Sie Andere einladen. Geben Sie die E-Mail-Adressen der Benutzer oder der externen Partner ein. Optional geben Sie eine Nachricht für die Empfänger ein.

    • Um einen Link zu erhalten, den Sie zum Beispiel per E-Mail, Messenger oder in sozialen Netzwerken weitergeben können, wählen Sie Get a link.

  4. Standardmäßig werden die Daten auf unbestimmte Dauer zum Lesen freigegeben. Um ein andere Art der Freigabe festzulegen, klicken Sie auf Erweiterte Optionen. Aktivieren Sie eine oder mehrere der folgenden Optionen.

    • Wenn die Empfänger der Freigabe die Daten auch bearbeiten sollen, aktivieren Sie Empfänger können bearbeiten.

    • Wenn Sie den Zugriff mit Hilfe eines Passworts schützen wollen, aktivieren Sie Passwort erforderlich. Geben Sie ein Passwort ein. Merken Sie sich das Passwort.

    • Wenn die Freigabe nach einer bestimmten Zeit erlöschen soll, aktivieren Sie Läuft ab in. Wählen Sie einen Zeitraum.

  5. Klicken Sie auf Teilen.
