8.6.6. Aufgaben drucken

Um Aufgaben zu drucken, haben Sie folgende Möglichkeiten:

  • die Daten einer Aufgabe drucken

So drucken Sie die Daten einer Aufgabe:

  1. Wählen Sie in der Liste eine Aufgabe.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie im Menü auf Drucken. Ein Fenster mit einer Druckvorschau wird geöffnet.

  3. Falls erforderlich, ändern Sie die Druckeinstellungen. Klicken Sie auf die Schaltfläche Drucken.

  4. Schließen Sie das Fenster mit der Druckvorschau.

Tipp: Sie können auch die Daten mehrerer Aufgaben gemeinsam drucken.
