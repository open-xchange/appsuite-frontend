5.6.4. Adressen sammeln

Sie können E-Mail-Adressen sammeln, indem Sie die Adressen zu einem Adressbuch hinzufügen. Sie haben folgende Möglichkeiten:

  • Neue E-Mail-Adressen beim Senden oder Lesen von E-Mails automatisch sammeln, indem Sie diese Funktion in den E-Mail-Einstellungen aktivieren.

  • E-Mail-Adressen manuell in ein Adressbuch einfügen

So fügen Sie eine E-Mail-Adresse manuell in ein Adressbuch ein:

  1. Wählen Sie eine E-Mail.

  2. Klicken Sie in der Detailansicht auf den Namen des Absenders oder eines Empfängers.

    Klicken Sie im Popup auf Zum Adressbuch hinzufügen.

Hinweis: Diese Funktion ist nur bei neuen Kontakten verfügbar.

Verwandte Themen:

  • Einstellungen zum Empfangen von E-Mails

  • Neuen Kontakt anlegen
