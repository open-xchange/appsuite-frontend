8.6.4. Aufgaben verschieben

Sie können eine Aufgabe oder mehrere Aufgaben gemeinsam in einen anderen Ordner verschieben.

So verschieben Sie eine Aufgabe:

Voraussetzung: Sie haben für den Zielordner die Berechtigung zum Anlegen von Objekten.

  1. Wählen Sie in der Liste eine Aufgabe.

  2. Klicken Sie in der Werkzeugleiste auf das Symbol Aktionen . Klicken Sie im Menü auf Verschieben.

  3. Wählen Sie im Fenster Verschieben einen Ordner. Klicken Sie auf OK.

Tipp: Um Aufgaben mit Hilfe von Drag and Drop zu verschieben, markieren Sie in der Liste eine Aufgabe oder mehrere Aufgaben. Ziehen Sie die markieren Aufgaben auf einen Ordner im Ordnerbaum.
