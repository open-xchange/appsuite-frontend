10.2.5. Einen Chat schließen

Um die Chat-Liste übersichtlich zu halten, können Sie Chats schließen und bei Bedarf wieder öffnen.

So schließen Sie einen Chat:

  1. Wählen Sie in der Chat-Liste einen privaten Chat, einen Gruppenchat oder einen öffentlichen Kanal.

  2. Klicken Sie rechts oben auf das Symbol Weitere Aktionen .

    Wählen Sie Chat schließen.

Der Chat und die Nachrichten im Chat-Feld werden geschlossen. Im Chat-Feld werden Symbole zum Erzeugen eines neuen Chats angezeigt.

So öffnen Sie einen geschlossenen Chat:

  1. Klicken Sie in der Chat-Liste auf Verlauf. Alle Chats werden im Chat-Feld angezeigt.

  2. Klicken Sie neben einem Chat auf Öffnen.
