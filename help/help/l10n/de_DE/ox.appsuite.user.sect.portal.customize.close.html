4.2.2. Portal-Widget entfernen

Sie können ein Widget vom Portal entfernen, indem Sie das Widget schließen.

So entfernen Sie ein Widget:

  • Klicken Sie im Widget auf das Symbol Schließen .

    Um das Widget wieder anzuzeigen, verwenden Sie die Portal-Einstellungen.
