8.6.7. Aufgaben löschen

Sie können eine Aufgabe oder mehrere Aufgaben gemeinsam löschen.

So löschen Sie eine Aufgabe:

Achtung: Wenn Sie eine Aufgabe löschen, ist diese Aufgabe unwiederbringlich verloren.

  1. Wählen Sie in der Liste eine Aufgabe.

  2. Klicken Sie in der Werkzeugleiste auf Löschen.

  3. Bestätigen Sie, dass Sie die Aufgabe löschen möchten.

Ergebnis: Die Aufgabe wird gelöscht.
