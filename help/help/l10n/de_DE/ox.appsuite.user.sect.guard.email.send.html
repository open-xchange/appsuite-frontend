12.2.2. Verschlüsselte E-Mails senden

Sie haben folgende Möglichkeiten:

  • Eine E-Mail verschlüsselt senden. Nur Sie und die Empfänger können den Inhalt der E-Mail lesen.

  • Eine E-Mail mit Signatur senden. Die Signatur stellt sicher, dass der Empfänger erkennen kann, ob der E-Mail Inhalt auf dem Transportweg verändert wurde.

  • Eine E-Mail verschlüsselt und mit Signatur senden.

So senden Sie eine verschlüsselte E-Mail:

  1. Im E-Mail-Bearbeitungsfenster klicken Sie in der Schaltflächenleiste auf das Symbol Verschlüsselung aktivieren . Oberhalb des E-Mail-Textes wird eine Funktionsleiste angezeigt.

    Symbole neben den Empfängern zeigen an, ob die Nachricht für diesen Empfänger verschlüsselt werden kann. Wenn Sie auf ein Symbol zeigen, wird eine Erläuterung angezeigt.

  2. Um die E-Mail zusätzlich zu signieren, klicken Sie in der Funktionsleiste auf Optionen . Aktivieren Sie Signieren.

  3. Wenn in den Guard-Einstellungen Erweiterte Einstellungen anzeigen aktiviert ist, können Sie weitere Optionen verwenden. Dazu klicken Sie in der Funktionsleiste auf Optionen. Aktivieren Sie bei Bedarf die folgenden Optionen:

    • Standardmäßig werden verschlüsselte E-Mails im Format PGP Mime gesendet. Einige ältere E-Mail Clients unterstützen nur PGP Inline. Wenn der Empfänger einen solchen Client verwendet, aktivieren Sie PGP Inline.

      Hinweis: Wenn Sie PGP Inline verwenden, können Sie keine E-Mails im HTML-Format senden.

    • Damit der Empfänger die E-Mail verschlüsselt beantworten kann, benötigt er Ihren öffentlichen Schlüssel. Bei Bedarf können Sie Ihren öffentlichen Schlüssel als Anhang senden. Dazu aktivieren Sie Meinen Schlüssel anhängen.

  4. Klicken Sie auf Senden.

    Wenn Sie an neue Empfänger senden, wird ein Fenster angezeigt, in dem Sie den Empfängern Hinweise zum Öffnen der verschlüsselten E-Mail senden können.
