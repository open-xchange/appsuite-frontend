9.6. Freigaben

Mit Hilfe von Freigaben können Sie mit anderen Benutzern oder mit externen Partnern zusammenarbeiten, indem Sie in der App Drive Dateien oder Ordner zum Lesen oder Bearbeiten zur Verfügung stellen. Sie haben die folgenden Möglichkeiten.

  • Dateien oder Ordner für andere Benutzer oder für externe Partner zum Lesen oder Bearbeiten freigeben.

  • Auf Dateien oder Ordner zugreifen, die andere Benutzer für Sie freigeben.

  • Verschaffen Sie sich einen Überblick über Ihre Freigaben, indem Sie diese auf einer Übersichtsseite anzeigen.

  • Verwalten Sie Ihre Freigaben, indem Sie neue Benutzer hinzufügen, Benutzer entfernen oder Berechtigungen ändern.

Hinweis: Mit den nachfolgend beschriebenen Methoden können Sie Dateien und Ordner freigeben. Um E-Mails, Kontakte, Termine oder Aufgaben freizugeben, verwenden Sie die Methoden, die in 11.2: Berechtigungen beschrieben sind.
