Datenverschlüsselung mit Guard

12.6. Von Guard abmelden

Sie können sich von Guard abmelden, ohne die Groupware zu beenden. Um nach dem Abmelden eine verschlüsselte E-Mail oder Datei zu öffnen, müssen Sie das Guard-Sicherheitspasswort wieder eingeben.

Hinweis: Diese Funktion ist nur verfügbar, wenn Sie Passwort merken aktivieren, während Sie eine verschlüsselte E-Mail oder Datei öffnen.

So melden Sie sich von Guard ab:

  1. Klicken Sie in der Menüleiste rechts auf das Symbol Mein Konto.

  2. Klicken Sie auf Bei Guard abmelden.
