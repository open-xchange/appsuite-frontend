4.1.4. Widget Termine

Zeigt Ihre nächsten Termine. Sie können folgende Aktionen ausführen:

  • Wenn Sie auf einen Termin klicken, wird ein Popup geöffnet. Das Popup zeigt die Termindaten.

  • Um den Termin zu bearbeiten, klicken Sie im Popup auf die entsprechende Schaltfläche.

  • Anklicken eines Teilnehmers öffnet ein weiteres Popup.
