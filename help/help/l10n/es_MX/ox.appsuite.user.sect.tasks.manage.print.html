8.9.7. Impresión de tareas

Puede imprimir los datos de una sola tarea o de múltiples tareas.

Cómo imprimir los datos de una tarea:

  1. Seleccione una o varias tareas de la lista.

  2. Presione el icono Más acciones de la barra de herramientas. Presione en Imprimir. Se abrirá una ventana con una vista previa de impresión.

  3. Si es necesario, cambie los ajustes de la impresora. Presione el botón Imprimir.

  4. Cierre la ventana de vista previa de impresión.
