7.9.8. Exportación de citas

Puede exportar citas en formato iCalendar. Puede, por ejemplo, usar este formato para intercambiar citas con otras aplicaciones de calendario.Puede exportar citas individuales o todas las citas de un calendario.

Cómo exportar citas individuales:

  1. Dependiendo de la vista seleccionada, utilice las siguientes opciones:

    • Presione en una cita en una vista de calendario.

      En la ventana emergente, presione el icono Más acciones .

    • Seleccione una o varias citas en la vista de lista.

      Presione el icono Más acciones de la barra de herramientas.

    Presione Exportar. Se abre una ventana.

  2. Seleccione la función para guardar el archivo de exportación.

Cómo exportar todas las citas de un calendario:

  1. Seleccione una libreta de direcciones personal o pública en la vista de carpeta.

  2. Presione el icono Acciones que hay junto al nombre de la carpeta. Presione Exportar. Se abrirá una ventana.

  3. Seleccione la función para guardar el archivo de exportación.
