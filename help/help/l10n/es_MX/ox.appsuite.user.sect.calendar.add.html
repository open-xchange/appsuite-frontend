Notas:

  • Para crear una cita en un calendario, necesita tener permiso para crear objetos en el calendario.

  • No se pueden crear citas en los calendarios suscritos.

Consejo: Puede determinar si es mejor usar una cita o una tarea según los siguientes criterios:

  • Una cita tiene lugar en un punto definido en el tiempo. Si necesita ejecutar una acción en un punto definido en el tiempo, cree una cita para esta acción.

  • Una tarea tiene una fecha de vencimiento y, a veces, una prioridad. Si es flexible con respecto al cronograma de una acción y solo tiene que respetar la fecha de vencimiento, introduzca una tarea para esta acción.
